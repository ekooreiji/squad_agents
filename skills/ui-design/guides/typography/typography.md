## Recursos de Acessibilidade

### Tamanho Mínimo
- Texto principal: 16px
- Texto de links: 16px
- Labels: 14px

### Espaçamento
- Letter-spacing: -0.02em para maiúsculas
- Word-spacing: normal para texto

---

## Boas Práticas

- Limite a 2 famílias de fontes
- Use escala consistente
- Mantenha line-height adequada
- Garanta legibilidade
