# UI Design Skill

Skill especializada em design de interfaces de usuário, incluindo wireframes, mockups, protótipos e diretrizes de UI/UX.

---

## Visão Geral

Esta skill oferece guias, templates e exemplos para criar interfaces de usuário profissionais e acessíveis.

---

## Boas Práticas

- Sempre seguir princípios de UI/UX
- Garantir acessibilidade WCAG AA
- Usar cores com contraste adequado
- Manter consistência visual
- Testar em múltiplos dispositivos
