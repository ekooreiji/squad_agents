# Boas Práticas - Bulma

Boas práticas ao usar Bulma.

---

## Estrutura HTML

- Usar semantic markup
- section > container > columns > column
- Manter hierarquia

---

## Grid

- Máximo 12 colunas
- Não aninhar excessivamente
- Usar classes responsivas

---

## CSS

- Manter imports em ordem
- Não sobrescrever classes
- Usar variáveis Sass

---

## Performance

- Minificar CSS
- Usar only necessária
- CDN para produção

---

## Responsive

- Mobile-first
- Testar todos breakpoints
- is-12-mobile quando necessário

---

## Acessibilidade

- Labels visíveis
- semantic markup

---

## Cross-browser

- Testar Chrome, Firefox, Safari
- Flexbox support universal
