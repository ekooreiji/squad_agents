# Responsive Design Skill

Skill especializada em criar interfaces que se adaptam perfeitamente a qualquer dispositivo.

---

## Visão Geral

Esta skill cobre todo o sistema de design responsivo: de breakpoints estratégicos a testes cross-device.

## Diferença vs ui-design

| ui-design (atual) | responsive-design (nova) |
|-------------------|----------------------|
| Grid básico | Sistema completo |
| Breakpoints fixos | Múltiplos pontos |
| Tipografia fixed | Tipografia fluida |
| Não existe | Componentes responsivos |
| 124 linhas | 500+ linhas |
