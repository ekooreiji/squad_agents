# HTML/CSS Fundamentals Skill

Skill especializada em HTML e CSS básicos para designers que precisam colaborar com desenvolvedores.

---

## Visão Geral

Esta skill ensina os fundamentos de HTML e CSS que designers precisam saber para comunicar efetivamente com devs e entender código front-end.

---

## Guias por Categoria

### Fundamentos HTML
- `guides/html-basics.md` - Estrutura e tags semânticas
- `guides/forms.md` - Formulários HTML

### Fundamentos CSS
- `guides/css-basics.md` - Propriedades e valores
- `guides/box-model.md` - Modelo de caixa (margin, padding, border)
- `guides/selectors.md` - Seletores e specificity

### Layout
- `guides/flexbox.md` - Layout	flexível
- `guides/grid.md` - CSS Grid
- `guides/responsive-css.md` - Media queries

### Animações
- `guides/transitions.md` - Transições CSS
- `guides/animations.md` - Keyframes e animações

### Prática
- `guides/common-patterns.md` - Padrões comuns de UI
