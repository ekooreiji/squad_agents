# Boas Práticas - Pure.css

Boas práticas ao usar Pure.css.

---

## Módulos

- Importar apenas o necessário
- Usar CDN ou npm

---

## Grid

- Máximo 24 unidades
- Respects breakpoints

---

## Customização

- Sobrescrever classes CSS
- Manter minimalismo

---

## Performance

- Minificar em produção
- Apenas ~4KB total

---

## Acessibilidade

- semantic markup
- Labels claros

---

## Cross-browser

- Testar Chrome, Firefox, Safari
