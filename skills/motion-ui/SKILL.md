# Motion UI Skill

Biblioteca de animações da Foundation para transições e animações.

---

## Visão Geral

Motion UI oferece:

- Transições (fade, slide, zoom)
- Animações (hinge, shake, bounce)
- Toggle classes
- Magellan (scroll animado)
- Animações personalizáveis
