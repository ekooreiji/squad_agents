# Cores para Comunicação

Guia sobre como usar cores para transmitir mensagens de forma eficaz.

---

## 1. Significado das Cores

### 1.1 Cores Primárias e Seus Significados

| Cor | Significado | Emoção |
|-----|-------------|-------|
| Vermelho | Urgência, energia, peligro | Forte, intenso |
| Azul | Confiança, segurança, calma | Frio, profissional |
| Verde | Crescimento, saúde, natureza | Calmo, positivo |
| Amarelo | Atenção, otimismo, cautela | Alegre, alerta |
| Laranja | Entusiasmo, ação, calor | Vibrante |
| Roxo | Criatividade,luxo, mistério | Sofisticado |

### 1.2 Cores Neutras

| Cor | Significado | Uso |
|-----|-------------|-----|
| Preto | Poder, elegância | Texto,強調 |
| Branco | Pureza, clareza | Fundo, espaço |
| Cinza | Neutralidade | Secondary text |

---

## 2. Comunicação com Cores

### 2.1 Hierarquia de Cores

```
Cor Primária → Atenção primary
Cor Secundária → Support
Cor de Acent → Ação (CTAs)
Cor Neutra → Background
```

### 2.2 Significado Funcional

| Uso | Cor | Exemplo |
|-----|-----|---------|
| Alerta | Vermelho | Error, urgente |
| Sucesso | Verde | Confirmação |
| Info | Azul | Links,tips |
| Aviso | Amarelo/Laranja | Atenção |

---

## 3. Paleta para Comunicação

### 3.1 Paleta Mínima

```
Primária + Secundária + Acent + Neutra
   ↑          ↑          ↑        ↑
(1 cor)    (1-2 cores) (1 cor)  (cinzas)
```

### 3.2 Criando Significado

```
✓ Uma cor = Uma função

✓ Erro → Sempre vermelho
✓ Sucesso → Sempre verde
✗ Erro às vezes azul = Confusão
```

---

## 4. Aplicação Prática

### 4.1 Status Indicators

```
[🔴 Emergency] = Crítico
[🟡 Warning] = Alerta
[🟢 Success] = Bom
[🔵 Info] = Informativo
```

### 4.2 CTAs (Call to Action)

```
Botão primário = Cor de accent (destaque)
Botão secundário = Cor neutra/outline
```

### 4.3 Hierarchy Visual

```
Fundo claro → Conteúdo principal
Fundo médio → Secondary
Fundo escuro → Footer/contraste
```

## 5. Contraste e Legibilidade

### 5.1 Contraste WCAG

| Ratio | Level | Uso |
|-------|-------|-----|
| 3:1 | AA Large | Large text |
| 4.5:1 | AA | Body text |
| 7:1 | AAA | All text |

### 5.2 Combinações Seguras

| Fundo | Texto | Ratio |
|-------|-------|-------|
| White | #333 | 13:1 ✓ |
| White | #666 | 7:1 ✓ |
| #F5F5F5 | #666 | 4.5:1 ✓ |
| #333 | #666 | 2:1 ✗ |

## 6. Cores por Contexto

### 6.1 Redes Sociais

- Cores vibrantes = Mais engajamento
- Baixa saturação = Menos impacto

### 6.2 Apresentações

- Fundo branco = Mais legível
- Cores de accent = Destaque slides

### 6.3 Data Visualization

- Cores contrastantes = Diferenciação
- Evitar misleading
