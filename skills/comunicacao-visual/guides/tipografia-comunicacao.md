# Tipografia para Comunicação

Guia sobre como usar tipografia para comunicar mensagens com clareza.

---

## 1. O Papel da Tipografia

Tipografia é a voz visual do design.

---

## 2. Hierarquia Tipográfica

### 2.1 Sistema de Tamanhos

| Nível | Tamanho | Peso | Uso |
|-------|---------|------|-----|
| Display | 48-72px | Bold | Titles principales |
| H1 | 36-48px | Bold | Section titles |
| H2 | 28-32px | Semi-bold | Subtitles |
| H3 | 22-24px | Medium | Subsections |
| Body | 16-18px | Regular | Texto principal |
| Small | 14px | Regular | Secondary info |
| Caption | 12px | Regular | Notes |

### 2.2 Escalas

| Escala | Relação |
|--------|---------|
| Minor Third | 1.2 |
| Major Third | 1.25 |
| Perfect Fourth | 1.333 |
| Golden Ratio | 1.618 |

Formula: `Base × Ratio^n`

## 3. Escolhendo Fonts

### 3.1 Para Comunicação Clara

| Type | Recommendation | Use |
|------|--------------|-----|
| Sans-serif clean | Inter, Roboto, Open Sans | Digital |
| Humanist | Nunito, Raleway | Friendly |
| Geometric | Montserrat | Modern |

### 3.2 Limit

```
✓ Máximo 2-3 fontes por projeto
✓ 1 para headings
✓ 1-2 para body
```

## 4. Técnicas de Leitura

### 4.1.Line Length (Measure)

| Dispositivo | Caracteres por linha |
|-------------|---------------------|
| Mobile | 35-50 |
| Desktop | 45-75 |
| Otimo | 65 |

### 4.2 Line Height

| Type | Line Height |
|------|------------|
| Headings | 1.1-1.2 |
| Body | 1.4-1.6 |
| Large text | 1.3 |

### 4.3 Letter Spacing

| Type | Tracking |
|------|----------|
| CAPS titles | -0.5 a 0 |
| Body | 0 |
| Large dispay | 0.5-1 |

---

## 5. Legibilidade

### 5.1 Contrast

| Background | Text Color | WCAG |
|------------|-----------|-----|
| White | #333333 | ✓ AAA |
| White | #666666 | ✓ AA |
| #F5F5F5 | #999999 | ✗ Fail |

### 5.2 Size Minimo

| Context | Minimum |
|---------|---------|
| Body desktop | 16px |
| Body mobile | 14px |
| Captions | 12px |

## 7. Tipografia no Contexto

### 7.1 Apresentações

- Titles: Grande (40-60px)
- Bullets: Consistentes
- Espaço: Generoso

### 7.2 Redes Sociais

- Titles: Curto
- Corpo: Mínimo
- Hashtags: Pequeno

### 7.3 Infográficos

- Dados: Grande
- Context: Pequeno
- Fonts: System safe
