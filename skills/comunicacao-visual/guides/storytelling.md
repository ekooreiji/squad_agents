# Storytelling Visual

Guia sobre como construir narrativas visuais para comunicar mensagens.

---

## 1. O que é Storytelling Visual

Storytelling visual é a arte de comunicar uma mensagem através de uma narrativa visual coesa.

---

## 2. Estrutura Narrativa

### 2.1 Estrutura Básica

| Elemento | Descrição |
|----------|-----------|
| Início | Contexto, setup |
| Meio | Desenvolvimento, conflito |
| Fim | Resolução, call-to-action |

### 2.2 Estrutura de Presentations

```
1.-hook ( Atenção)
   ↓
2. Problem ( Desafio)
   ↓
3. Solution ( Solução)
   ↓
4. benefit (Benefício)
   ↓
5. CTA (Ação)
```

---

## 3. Elementos da Narrativa

### 3.1 Person

```
✓ Person real/relacionável
✓ Identificável pelo público
✓ Com objetivos claros
```

### 3.2 Plot

```
Setup → Conflict → Climax → Resolution
```

### 3.3 Purpose

```
Qual a mensagem?
Por que importa?
O que fazer?
```

---

## 4. Técnicas Visuais

### 4.1 Sequential Flow

```
[Etapa 1] → [Etapa 2] → [Etapa 3]
     →          →           →
```

### 4.2 Before/After

```
ANTES          DEPOIS
[ Problema ] → [ Solução ]
```

### 4.3 Comparison

```
[A]     vs     [B]
[Original]     [Novo]
```

## 6. Exemplos de Narrativas

### 6.1 Transformação

```
1. [Foto antes]
2. [Processo]
3. [Foto depois]
Narrativa: "De X para Y"
```

### 6.2 Jornada

```
1. [Desafio]
2. [Decisão]
3. [Ação]
4. [Resultado]
```

### 6.3 Problema → Solução

```
1. [Estatística do problema]
2. [Nossa solução]
3. [Resultado]
```

---

## 7. Boas Práticas

### 7.1 Clareza

- Uma mensagem por seção
- Language simples
- Visual direto

### 7.2 Engajamento

- Criar curiosidade
- Mostrar valor
- Facilitar ação

### 7.3 Consistência

- Mesmo estilo visual
- Mesmo tom
- Mesma paleta
