# Ideação em Design Thinking

Guia sobre brainstorming e geração de ideias.

---

## 1. Preparação

### 1.1 Antes de Idear

- Problema definido
- Insights compilados
- HMW definidos
- Materiais prontos

### 1.2 Regras do Brainstorm

| Regra | Descrição |
|-------|-----------|
| No judgment | Sem críticas |
| Go for quantity | Quantity > Quality |
| Build on ideas | Usar ideias dos outros |
| Stay focused | Manter no tema |
| One conversation | Uma pessoa por vez |

---

## 2. Técnicas de Ideação

### 2.1 Brainstorming Tradicional

1. Definir tema
2. Timer (10-20 min)
3. Ideas individuais
4. Compartilhar em round
5. Combinar/agrupar

### 2.2 Brainstorming Inverso

```
1. Começar com pior ideia possível
2. Por que ela não funciona
3. Mudar para funcionar
4. Iterar até idea boa
```

### 2.3 SCAMPER

| Letra | Pergunta | Exemplo |
|-------|---------|---------|
| S | Substituir? | Novo material |
| C | Combinar? | Agregar feature |
| A | Adaptar? | Outro contexto |
| M | Modificar? | Mudar tamanho |
| P | Put to other use? | Novo uso |
| E | Eliminar? | Remover steps |
| R | Reverse? | Inverter processo |

## 4. Ideação em Grupo

### 4.1 Estrutura

| Fase | Tempo | Atividade |
|------|-------|-----------|
| Warm-up | 5 min | Game rápido |
| Geração | 10 min | Silent write |
| Compartilhar | 10 min | Round robin |
| Clustering | 5 min | Agrupar ideas |

### 4.2多人 Ideação

- Silent storm (escrito)
- Round robin (verbal)
- Chalktalk (parede)
- Sticky notes (visual)

---

## 5. Filtrar Ideias

### 5.1 Dotmocracy

```
Cada participante tem 3 dots
Colocar em ideas favoritas
Ideas mais votadas =prioritárias
```

### 5.2 Critérios de Seleção

| Critério | perguntas |
|----------|-------------|
| Viabilidade | Dá para fazer? |
| Impacto | Resolve problema? |
| Usuário | Usuários querem? |

### 5.3 Matriz

```
Alto impacto + Baixo esforço = Quick wins
Alto impacto + Alto esforço = Estratégico
Bajo impacto + qualquer = Eliminar
```

## 6. Documentation

### 6.1O que Documentar

- Todas as ideias
- Nome de participantes
- Contexto original
- Decisões de filtragem

### 6.2Template de Idea

```
IDEA: [Nome/título]
DESCRIÇÃO: [O que faz]
PARA: [Usuário]
QUANDO: [Contexto]
BENEFÍCIO: [Valor]
```
