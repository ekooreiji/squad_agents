# Design Thinking

Skill especializada em abordagem centrada no usuário para resolver problemas complexos.

---

## Visão Geral

Esta skill ensina a:

- Aplicar empatia para entender usuários
- Definir problemas corretamente
- Gerar ideias criativas
- Prototipar rapidamente
- Testar e iterar
- Usar ferramentas e métodos
