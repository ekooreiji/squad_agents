# Publicação - Flutter

Publicar apps Flutter.

---

## Android

```bash
flutter build apk --debug
flutter build apk --release
```

---

## iOS

```bash
flutter build ios --simulator --no-codesign
flutter build ios --release
```

---

## Web

```bash
flutter build web
```

---

## App Store (iOS)

1. Criar Apple Developer account
2. Configurar no Xcode
3. Publicar via App Store Connect

---

## Google Play (Android)

1. Criar Google Play Console
2. Gerar signed APK
3. Fazer upload
