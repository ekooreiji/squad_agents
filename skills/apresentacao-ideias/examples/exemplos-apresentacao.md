# Exemplos de Apresentação

Exemplos práticos de apresentações de ideias.

---

## Exemplo 1: Presentation de Projeto

### Contexto
Reunião com cliente para apresentar redesign de website.

### Abertura

```
"Bom dia! Hoje vamos apresentar
a proposta de redesign para
[Cliente]. Vamos começar
com o contexto..."
```

### Problema

```
" Pesquisamos e descobrimos que
70% dos usuários abandonam
na página de checkout.

Os principais motivos são:
- Processo muito longo
- Falta de confiança
- Não acham o que procuram"
```

### Solução

```
"Nossa proposta resolve isso
através de:
1. Checkout em 3 passos
2. Trust badges visíveis
3. Busca melhorada"
```

### Benefícios

```
"O resultado esperado:
- 35% menos abandono
- 25% mais conversão
- NPS acima de 50"
```

### CTA

```
"Precisamos de aprovação
até sexta para iniciar.
Alguma questão?"
```

## Exemplo 2: Elevator Pitch

### Situação
Em um evento, alguém pergunta sobre seu trabalho.

### Resposta (45 segundos)

```
"Somos TechLearn, uma
plataforma de capacitação
corporativa.

[Problema] Empresas gastam
fortune em treinamento
que não gera resultado.

[Solução] Nossa metodologia
usa microlearning +
gamificação com
resultado 3x melhor.

[Diferencial] Medimos
resultado em ROI,
não em horas.

[CTA] Vou te enviar
um case de sucesso."
```

## Exemplo 3: Investor Pitch

### Estrutura

```
Slide 1:Title - EasyCart
Slide 2: Problema - Checkout problems
Slide 3: Solução - One-click checkout
Slide 4: Mercado - TAM R$ 10B
Slide 5: Modelo - SaaS + transaction
Slide 6: Traction - 100 clients, 35% grown
Slide 7: Competição - X vs nós
Slide 8: Equipe - Experts
Slide 9: Ask - R$ 2M
```

### Key Points

- Dados em vez de opiniões
- Crescimento consistente
- Equipe forte
- ask claro

## Exemplo 4: Respondendo Objeção

### Situação
Cliente diz "É muito caro".

### Resposta

```
"Entendo sua preocupação
sobre investimento.

[Explore] Quando você
diz caro, está falando
de...?

1. Custo inicial?
2. ROI?
3. Comparativo com outra?"

[Se for orçamento]
"O investimento se paga
em 4 meses.Vamos
calcular junto."
```

## Exemplo 5: Justificativa de Design

### Situação
Por que o botão é azul?

```
"Porque azul é a cor da
marca (documentation).
Pesquisas mostram que
azulpassa confiabilidade.
Contraste WCAG AAA.
Usamos também em
todos os CTAs para
consistência."
```
