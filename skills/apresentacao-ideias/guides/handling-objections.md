# Lidando com Objeções

Guia sobre como responder objeções durante apresentações.

---

## 1. Tipos de Objeções

### 1.1 Por Origem

| Tipo | O que significa |
|------|----------------|
| Técnica | Não entende como funciona |
| Financeira | Caro/ sem orçamento |
| Timing | Não é agora |
| Política | Não tem apoio |
| Competição | Tem outra opção |

### 1.2 Por Intenção

| Tipo | Como identificar |
|------|----------------|
| Genuína | Quer entender melhor |
| Política | Quer mudar o assunto |
| blockers | Já decidiu não |

## 2. Framework de Resposta

### 2.1 Framework LAER

```
L - Listen (Ouvir)
A - Acknowledge (Reconhecer)
E - Explore (Explorar)
R - Respond (Responder)
```

### 2.2应用例

```
Ojeção: "Isso é muito caro"

Acknowledge: "Entendo sua preocupação..."

Explore: "Quando você diz caro,
você está preocupado com...?"
- Preço inicial?
- ROI?
- Custo de implementação?

Respond: "O investimento se paga
em 4 meses porque..."
```

## 3. Respostas por Tipo

### 3.1 "É muito caro"

```
✓ "Compared ao custo de não fazer,
o custo de fazer é menor"
✓ "Qual seria seu orçamento?"
✓ "Vamos calcular o ROI juntos"
```

### 3.2 "Não é prioridade"

```
✓ "O que é prioridade agora?"
✓ "E se não resolver, o que acontece?"
✓ "Quanto custa não resolver?"
```

### 3.3 "Já temos uma solução"

```
✓ "O que vocês usam hoje?"
✓ "O que está funcionando bem?"
✓ "O que poderia melhorar?"
```

### 3.4 "Timing não é bom"

```
✓ "Quando seria um bom timing?"
✓ "Qual o impacto de esperar?"
✓ "O que acontece se esperarmos?"
```

## 4. Técnicas de Persuasão

### 4.1 Feel-Felt-Found

```
"Entendo como você se sente.
Outros já se sentiram assim.
O que eles encontraram foi..."
```

### 4.2 Agree-Answer-Advance

```
Agree: "Você está certo sobre X"
Answer: "A razão para isso é Y"
Advance: "E além disso..."
```

### 4.3 Redirect

```
"Nãobem. Mais importante que X,
temos que pensar em Y porque..."
```

---

## 5. Turning objectors into Champions

### 5.1 Processo

```
1. Objeção genuína → Pergunta sincera
2. Resposta satisfatória → Aliado
3. Objeção respondida → Advocacy
```

### 5.2 Exemplo

```
Cliente: "Não sei se isso funciona"
→ Pergunta: "Qual sua maior dúvida?"
→ Resposta completa
→ Cliente: "Vamos tentar!"
```

## 6. Técnicas Defensivas

### 6.1 Acknowledge Question

```
✓ "Boa pergunta"
✓ "Entendo"
✓ "Isso é importante considerar"
```

### 6.2 Defer

```
"Essa é uma discussão mais profunda.
Podemos abordar offline?"
```

### 6.3 Bridge

```
"Isso se conecta ao que
mencionamos sobre X..."
```

## 7. Pré-planejar Objeções

### 7.1 Antecipar

- O que听众 vai objetar?
- Por que vão resistir?
- Qual seu ponto fraco?

### 7.2 Preparar Respostas

| Possible Objeção | Resposta Preparada |
|------------------|-------------------|
| Preço | Cálculo ROI |
| Timeline | Versão mínima viável |
| Recursos | Escala gradual |
