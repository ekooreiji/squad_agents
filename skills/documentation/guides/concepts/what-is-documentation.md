# O que é Documentação

## Definição

Documentação é qualquer informação que descreve um sistema, suas funcionalidades, ou como usá-lo.

## Tipos de Documentação

| Tipo | Descrição | Público |
|------|-----------|---------|
| **Código** | README, docstrings | Desenvolvedores |
| **Arquitetura** | ADRs, diagramas | Tech leads |
| **Usuário** | Manuais, guias | Usuários finais |
| **Operacional** | Deploy,运维 | DevOps/infra |

## Por que Documentar

| Benefício | Descrição |
|-----------|-----------|
| **Onboarding** | Novos devs entendem mais rápido |
| **Manutenção** | Facilita manutenções futuras |
| **Comunicação** | Alinha expectativas |
| **Memória** | Preserva conhecimento |
