# Checklist React

## Estrutura

- [ ] Create React App ou Vite configurado
- [ ] Estrutura de pastas definida
- [ ] Componentes organizados por feature
- [ ] Assets (imagens, estilos) separados

## Componentes

- [ ] Componentes funcionais (não classes)
- [ ] Props tipadas (TypeScript opcional)
- [ ] Componentes puros (sem estado desnecessário)
- [ ] Nomes claros e descritivos

## State

- [ ] useState para estado local
- [ ] State lifting para estado compartilhado
- [ ] Context para estado global
- [ ] Evitar estado redundante

## Hooks

- [ ] useEffect com dependency array correto
- [ ] useCallback para callbacks em props
- [ ] useMemo para cálculos pesados
- [ ] Regras dos hooks seguidas

## Performance

- [ ] Componentes puros
- [ ] Memoização quando necessário
- [ ] Lazy loading de rotas
- [ ] Keys únicas em listas

## Estilos

- [ ] CSS Modules ou styled-components
- [ ] Variáveis CSS para temas
- [ ] Responsividade
- [ ] Acessibilidade (ARIA)

## Testing

- [ ] Jest configurado
- [ ] React Testing Library
- [ ] Testes unitários de componentes
- [ ] Testes de integração

## Rotas

- [ ] React Router configurado
- [ ] Rotas protegidas
- [ ] Parâmetros validados
- [ ] Navegação funcional

## Boas Práticas

- [ ] ESLint configurado
- [ ] Prettier configurado
- [ ] Conventional commits
- [ ] Documentação básica
