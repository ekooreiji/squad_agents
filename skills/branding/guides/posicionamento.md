# Posicionamento de Marca

Guia sobre posicionamento estratégico de marca.

---

## 1. O que é Posicionamento

Posicionamento é o lugar que sua marca ocupa na mente do consumidor em relação aos concorrentes.

---

## 2. Framework de Posicionamento

### 2.1 Posicionamento Statement

```
Para [target] que [necessidade],
[nome da marca] é [categoria]
que [benefício/key differentiation].

Diferente de [concorrentes],
[nome da marca] [reason to believe].
```

### 2.2 Exemplo

```
Para devs que precisam de trabalho,
Nubank é banco digital
que simplifica banking.

Diferente de bancos tradicionais,
Nubank oferece experiência sem burocracia.
```

## 3. Análise Competitiva

### 3.1 Matriz de Posicionamento

| | Preço Alto | Preço Baixo |
|---|---|---|
| Qualid. Alta | Premium | Value |
| Qualid. Baixa | Overpriced | Budget |

### 3.2 Mapa perceptual

```
         Inovação →
    A        B
    ↕          ↕
  Qualid.      C
    ↕          ↕
        Preço →
```

---

## 4. Diferenciação

### 4.1 Tipos

| Tipo | Como |
|------|------|
| Produto | Features, tecnologia |
| Serviço | Experiência, suporte |
| Preço | Valor, economia |
| Experiência | Lifestyle, comunidade |

### 4.2 Unique Selling Proposition (USP)

```
O que NOS fazer de único
que outros NÃO podem replicar
e que clientes VALORIZAM
```

---

## 5. Target

### 5.1 Definição

| | Descrição |
|---|---|
| Primário | Main audience |
| Secundário | Next audience |
| Tercário | Awareness |

### 5.2 Segmentos

- Demográfico (idade, gênero)
- Psicográfico (valores)
- Comportamental (uso)
- Geográfico (local)

---

## 6. Proposta de Valor

### 6.1 Proposta deValor

```
PROPOSTA: [Benefício principal]
PROBLEMA: [Dor resolvida]
SOLUÇÃO: [Como resolve]
```

### 6.2 Benefício Types

| Tipo | Exemplo |
|------|--------|
| Funcional | Mais rápido |
| Emocional | Segurança |
| Social | Status |
| Transformacional | Crescimento |
