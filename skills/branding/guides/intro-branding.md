# Introdução ao Branding

Guia introdutório sobre branding.

---

## 1. O que é Marca

Marca é o conjunto de percepções, experiências e associações que as pessoas têm sobre uma empresa, produto ou serviço.

---

## 2. Definições de Marca

### 2.1 Marca vs Logo

| Conceito | Definição |
|---------|-----------|
| Marca | Percepção total |
| Logo | Identidade visual |
| Identidade | O que você é |
| Imagem | O que outros veem |

### 2.2 Brand Equity

```
Brand equity = Valor adicional da marca
- Awareness (Reconhecimento)
- Associations (Associações)
- Percepções (Percepções)
- Comportamentos (Comportamentos)
```

## 6. Processo de Branding

### 6.1 Etapas

```
1. DESCOBERTA  → Entender contexto
2. ESTRATÉGIA → Definir posicionamento
3. DESIGN   → Criar identidade
4. VOZ     → Definir comunicação
5. GOVERNANÇA → Management
6. TOOLKIT  → Documentar
```

### 6.2Timeline

| Fase | Tempo |
|------|-------|
| Descoberta | 1-2 sem |
| Estratégia | 2-4 sem |
| Design | 4-8 sem |
| Total | 2-4 meses |
