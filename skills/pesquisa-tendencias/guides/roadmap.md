# Roadmap de Tendências

Guia sobre como criar roadmap de inovação baseado em tendências.

---

## 1. O que é Roadmap

Roadmap de tendências é um plano de implementação de tendências ao longo do tempo.

---

## 2. Estrutura

### 2.1 Dimensões

| Eixo | O que representa |
|------|-----------------|
| Horizontal | Tempo (trimestres) |
| Vertical | Categoria/Ouadrante |
| Tamanho | Impacto/Esforço |

### 2.2 Template visual

```
Q1          Q2          Q3          Q4
|----------|----------|----------|----------|
| [Tren 1] |          |           |          |
|    [T2]  |  [T3]    |           |          |
|          |    [T4]  |  [T5]     |          |
```

---

## 3. Criar Roadmap

### 3.1 Passo a Passo

1. Listar tendências validadas
2. Priorizar por impacto
3. Definir timing
4. Atribuir recursos
5. Documentar dependencies

### 3.2 Critérios

| Critério | Peso |
|---------|------|
| Impacto no usuário | Alto |
| Viabilidade | Médio |
| Diferenciação | Médio |

---

## 4. Categorias

### 4.1 por Área

```
UI/ UX
   └── Components
Tecnologia
    └── Features
Marketing
    └── Campanhas
Produto
    └── Roadmap
```

### 4.2por Status

```
Now      → Implementando
Next     → Próximo
Later    → Futuro
Never    → Descartado
```

---

## 5. Manter Roadmap

### 5.1 Review

| Frequência | O que revisar |
|------------|--------------|
| Mensal | Progress |
| Trimestral | Priorities |
| Anual | Estratégia |

### 5.2 Update Triggers

- Nova tendência significativa
- Mudança de mercado
- Feedback de resultado
