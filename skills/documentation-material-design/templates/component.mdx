## Usage Example

<CodeBlock
  code={`import { Button } from '@mui/material'
import SaveIcon from '@mui/icons-material/Save'

function Example() {
  return (
    <Button
      variant="contained"
      color="primary"
      size="large"
      startIcon={<SaveIcon />}
      onClick={() => console.log('clicked')}
    >
      Save
    </Button>
  )
}`}
/>

<TipBox title="Tip">
  Use startIcon for primary actions to provide visual emphasis.
</TipBox>

<WarningBox title="Warning">
  Avoid using too many buttons with the same color in the same area.
</WarningBox>

## Accessibility

- Use meaningful labels
- Ensure sufficient color contrast
- Focus states are visible
