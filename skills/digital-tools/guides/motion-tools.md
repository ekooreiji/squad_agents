## 1. Ferramentas Professionais

### 1.1 Adobe After Effects

| Recurso | Descrição |
|---------|-----------|
| melhor para | Motion graphics, compositing |
| Plataforma | macOS, Windows |
| Custo | Subscription |

**Atalhos Essenciais:**

| Atalho | Função |
|--------|--------|
| Space | Play/Pause |
| J/K/L | Rewind/Forward/Pause |
| N | End of work area |
| I | Start of work area |
| P | Position keyframes |
| S | Scale keyframes |
| R | Rotation keyframes |
| U | Show only keyframes |
| G | Track matte |
