# Collaboration Tools

Guia de ferramentas para colaboração em projetos digitais.

---

## 1. Ferramentas de Design Collaboration

### 1.1 Frame.io

| Recurso | Descrição |
|---------|-----------|
| melhor para | Video review collaboration |
| Integração | Premiere, After Effects |
| Features | Comments, timestamps,Versions |

### 1.2 Zeplin

| Recurso | Descrição |
|---------|-----------|
| melhor para | Design handoff |
| Features | Specs, assets, code snippets |

### 1.3 Avocode

| Recurso | Descrição |
|---------|-----------|
| melhor para | Design hand-off |
| Features | CSS, iOS, Android code |

---

## 2. Ferramentas de Planejamento

### 2.1 Miro

| Recurso | Descrição |
|---------|-----------|
| melhor para | Brainstorming, workshops |
| Features | Sticky notes, diagrams, templates |

### 2.2 Notion

| Recurso | Descrição |
|---------|-----------|
| melhor para | Documentação, project management |
| Features | Databases, wikis, docs |

### 2.3 Linear

| Recurso | Descrição |
|---------|-----------|
| melhor para | issue tracking, projetos |
| Features | Projects, sprints, epics |

---

## 3. Fluxo de Trabalho Colaborativo

### 3.1 Design → Development

```
Figma → (comments/versions) → Zeplin → Code
         ↓
    Frame.io (assets) → Handoff
```

### 3.2 Project Management

```
Notion → (documentation) → Linear → (tracking)
```
