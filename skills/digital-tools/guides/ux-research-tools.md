## 4. Fluxo de Pesquisa

### 4.1 Processo

```
1. Define objectives
   │ Research questions
   │
2. Choose method
   │ Survey vs Test vs Interview
   │
3. Select tool
   │ Maze, UserTesting, Typeform
   │
4. Conduct
   │ Recruit participants
   │ Execute
   │
5. Analyze
   │ Reports
   └── Synthesize insights
```
