# Motion Design Skill

Skill especializada em motion design, animações para UI e vídeos para redes sociais.

---

## Visão Geral

Esta skill cobre todo o espectro de motion design: de micro-interações em UI a vídeos curtos para Instagram, TikTok e outras plataformas.
