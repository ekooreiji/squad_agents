# Video Editing

Guia básico de edição de vídeo para motion design.

---

## 1. Conceitos Básicos

### 1.1 Timeline

```
| Clip 1 | Clip 2 | Clip 3 | Transit | | Clip 4 |
```

### 1.2 Estrutura de Corte

| Tipo | Uso | Duração |
|------|-----|----------|
| **J-cut** | Audio avança do vídeo | - |
| **L-cut** | Vídeo avança do áudio | - |
| **Cut direto** | Transição simples | 0-1s |
| **Dissolve** | Transição suave | 1-3s |

---

## 2. Transições

## 3. Efeitos Common

### 3.1 Transpadrões

| Efeito | Uso |
|--------|-----|
| Fade | Alternância de cena |
| Zoom | Destaque de elemento |
| Glitch | Efeito moderno |
| Blur | Transição suave |

### 3.2 Speed

| Efeito | CSS (equivalent) |
|--------|------------------|
| Slow-mo | 0.5x speed |
| Speed-up | 2x speed |
| Freeze frame | hold frame |

| Item | Verificar |
|------|----------|
