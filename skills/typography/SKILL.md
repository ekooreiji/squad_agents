# Typography Skill

Skill especializada em tipografia, cobrindo escolha, combinação e sistema tipográfico para projetos digitais e impressos.

---

## Visão Geral

Domine a arte de escolher e combinar fontes de forma eficaz. Esta skill vai além do básico de "quais fontes usar" para explicar "como escolher" e "como combinar".
