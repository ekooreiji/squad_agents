# Color System for Brand

Guia para definir sistema de cores de marca.

---

## 1. Estrutura de Cores

### 1.1 Categorias

| Categoria | Uso | Exemplos |
|-----------|-----|----------|
| **Primary** | Marca principal | Logo, CTAs |
| **Secondary** | Suporte | Elementos secundários |
| **Accent** | Destaques | Hot-links, badges |
| **Neutral** | Fundos, texto | Backgrounds |
| **Functional** | Feedback | Success, error, warning |

### 1.2 Proporções

```
┌─────────────────────────────────────────────┐
│                                             │
│  Primary     → 60% (uso principal)         │
│  Secondary   → 30% (suporte)             │
│  Accent      → 10% (destaque)            │
│                                             │
└─────────────────────────────────────────────┘
```
