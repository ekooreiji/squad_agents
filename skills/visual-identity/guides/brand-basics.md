# Brand Basics

Guia completo de fundamentos de marca, naming e posicionamento.

---

## 1. O que é Marca

Marca é a percepção que as pessoas têm de uma empresa, produto ou serviço. Vai além do logotipo.

---

## 2. Elementos de Marca

### 2.1 Marca vs Logotipo

| Conceito | Definição |
|----------|----------|
| **Marca** | Percepção, emoção, reputação |
| **Logotipo** | Identidade visual, símbolo |

### 2.2 Componentes da Marca

```
┌─────────────────────────────────────────────────┐
│                    MARCA                        │
├─────────────────────────────────────────────────┤
│  Nome           → como chamada                    │
│  Tagline        → como comunicada              │
│  Posicionamento → como diferenciada            │
│  Personalidade → como expressa                │
│  Valores        → o que representa           │
│  Identidade     → como visualizada           │
└─────────────────────────────────────────────────┘
```

## 3. Naming (Nome da Marca)

### 3.1 Tipos de Nomes

| Tipo | Exemplo | Características |
|------|---------|-----------------|
| Descritivo | IBM | Explícito, claro |
| Afetivo | Apple | Emocional, acessível |
| Acrónimo | HP | Breve, memorável |
| Neologismo | Google | Único, moderno |
| Metáfora | Amazon | Evocativo |

### 3.2 Processo de Naming

```
PASSO 1: Definir atributos da marca
          │
          ▼
PASSO 2: Gerar nomes (brainstorm)
          │
          ▼
PASSO 3: Filtrar nomes
          │
          ▼
PASSO 4: Verificar disponibilidade
          │
          ▼
PASSO 5: Testar com público
          │
          ▼
PASSO 6: Selecionar final
```

### 3.3 Checklist de Nome

| Critério | Verificação |
|---------|-------------|
| Memorável | fácil de lembrar |
| Pronunciável | fácil de pronunciar |
| Disponível | domínio, TM |
| Relevante | relacionado ao negocio |
| Único | diferente dos concorrentes |
| Extensível | funciona para futuros produtos |

## 4. Posicionamento

### 4.1 Definição

Posicionamento é o lugar que a marca ocupa na mente do consumidor.

### 4.2 Modelo de Posicionamento

```
Para [público-alvo]
que precisa [benefício]
[nome da marca] é [categoria]
que [proposta de valor]

Exemplo:
Para profissionais de tecnologia
que precisam de soluções de design
Notion é uma ferramenta de produtividade
que conecta equipes em um único espaço
```

### 4.3 Mapa de Posicionamento

```
                    PREÇO ALTO
                        │
        ┌───────────────┼───────────────┐
        │               │               │
        │   PREMIUM     │   LUXO        │
        │               │               │
   ESCALA ─────────────┼────────────── ESCALA
        │               │               │
        │   ECONÔMICO  │   VALOR       │
        │               │               │
        └───────────────┼───────────────┘
                        │
                    PREÇO BAIXO
```

## 6. Tagline

### 6.1 Tipos de Tagline

| Tipo | Exemplo | Uso |
|------|---------|-----|
| Benefício | "Just Do It" | Inspiracional |
| Diferencial | "Think Different" | Posicionamento |
| Missão | "Organize the World" | Propósito |

### 6.2 Boas Práticas

- Máximo 5-7 palavras
- Memorável
- Relevante
- Atemporal
