## 3. Área de Segurança

### 3.1 Definição
A área de segurança é o espaço mínimo ao redor do logo.

```
┌─────────────────────────────────────┐
│      X      │    Logo     │    X     │
│             │             │          
└─────────────────────────────────────┘
X = 10% da largura do logo
```
