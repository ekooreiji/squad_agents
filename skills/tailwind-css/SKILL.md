# Tailwind CSS Skill

Skill especializada em Tailwind CSS para criar interfaces modernas e responsivas.

---

## Visão Geral

Esta skill ensina a:

- Configurar e customizar Tailwind CSS
- Criar componentes reuseis
- Implementar layouts com Flexbox e Grid
- Aplicar tipografia e cores
- Criar designs responsivos
- Adicionar animações
