# OWASP Prevention Checklist

## A01 - Broken Access Control

- [ ] Implementar autorização corretamente
- [ ] Denegar por padrão
- [ ] Verificar permissões em cada request

## A02 - Cryptographic Failures

- [ ] Dados sensíveis criptografados
- [ ] Algoritmos fortes
- [ ] Keys não hardcoded

## A03 - Injection

- [ ] Usar parameterized queries
- [ ] Escapar entradas
- [ ] ORM/framework

## A04 - Insecure Design

- [ ] Threat modeling
- [ ] Arquitetura segura
- [ ] Reviews de segurança

## A05 - Security Misconfiguration

- [ ] Hardening aplicados
- [ ] Configuração segura
- [ ] Updates regulares
