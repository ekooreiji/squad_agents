# Wireframing

Guia completo para criação de wireframes.

---

## 1. O que são Wireframes

Wireframes são representations visuais de baixa/média fidelidade da interface.

---

## 2. Níveis de Fidelidade

### 2.1 Baixa Fidelidade (Lo-Fi)

| Característica | Detalhe |
|--------------|----------|
| Desenho | Quick sketch (papel) |
| Cores | Preto/branco |
| Detalhes | Mínimo |
| Tempo | Minutos |
| Uso | Validação inicial |

### 2.2 Média Fidelidade (Mid-Fi)

| Característica | Detalhe |
|--------------|----------|
| Desenho | Ferramenta digital |
| Cores | Escala de cinza |
| Detalhes | Estrutura, sem estilo |
| Tempo | 1-2 horas |
| Uso | Revisão de stakeholders |

### 2.3 Alta Fidelidade (Hi-Fi)

| Característica | Detalhe |
|--------------|----------|
| Desenho | Figma/Sketch |
| Cores | Completo |
| Detalhes | near-final |
| Tempo | dias |
| Uso | handoff para dev |

## 3. Elementos do Wireframe

### 3.1 Layout

```
┌─────────────────────────────────────┐
│              HEADER                  │
├─────────────────────────────────────┤
│                                     │
│              HERO                   │
│                                     │
├─────────────────────────────────────┤
│          CONTENT GRID                │
│   ┌─────┐  ┌─────┐  ┌─────┐        │
│   │Card │  │Card │  │Card │        │
│   └─────┘  └─────┘  └─────┘        │
├─────────────────────────────────────┤
│              FOOTER                  │
└─────────────────────────────────────┘
```

### 3.2 Legenda

| Elemento | Significado |
|----------|-------------|
| [X] | Botão clicável |
| T | Texto |
| () | Imagem |
| ≡ | Menu |
| → | Link |

---

## 4. Boas Práticas

- Manter simples
- Focar em estrutura
- Não usar cores detalhadas
- Anotar interações
- Usar.grid Consistent
