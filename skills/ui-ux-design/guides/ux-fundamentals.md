# UX Fundamentals

Guia completo dos fundamentos de experiência do usuário (UX).

---

## 1. O que é UX

UX (User Experience) é a experiência completa que um usuário tem ao interagir com um produto ou serviço.

---

## 2. Diferença UI vs UX

| UI (User Interface) | UX (User Experience) |
|--------------------|---------------------|
| Visual | Comportamental |
| Aparência | Sensação |
| Componentes | Fluxos |
| Layout | Jornada |

---

## 3. Princípios UX

### 3.1 Heurísticas de Nielsen

| # | Heurística | Descrição |
|---|------------|----------|
| 1 | Visibilidade do status | Feedback constante |
| 2 | Correspondência | Igual ao mundo real |
| 3 | Controle e liberdade | Desfazer ações |
| 4 | Consistência | Padrões previsíveis |
| 5 | Prevenção de erros | minimizar erros |
| 6 | Reconhecimento | Lembrar = menos que recordar |
| 7 | Eficiência | Atalhos para experts |
| 8 | Design estético | Aparência agradável |
| 9 | Ajuda | Error recovery |
| 10 | Documentação | Help when needed |

### 3.2 Princípios Adicionais

| Princípio | Descrição |
|----------|----------|
| **Simplicidade** | Menos é mais |
| **Accessible** | Para todos |
| **Consistente** | Padrões previsíveis |
| **Eficiente** | Tarefas rápido |
| **Feedback** | Sempre informar |

## 4. Processo UX

### 4.1 Double Diamond

```
    ┌────────────────────────────────────────────────────────┐
    │                                                   │
    │   DISCOVER          DEFINE                        │
    │      ◇             ◇                           │
    │    ╱   ╲         ╱ ╲                          │
    │   ╱      ╲      ╱    ╲                         │
    │           ◇─────────◇                              │
    │              │                                 │
    │              │                                 │
    │           DEVELOP          DELIVER               │
    │              ◇             ◇                           │
    │           ╱   ╲         ╱ ╲                          │
    │          ╱      ╲       ╱    ╲                         │
    │                 ◇─────────◇                           │
    │                                                   │
    └────────────────────────────────────────────────────────┘
    
    DISCOVER: Pesquisa, entender
    DEFINE: Sintetizar, definir problema
    DEVELOP: Idear, prototipar
    DELIVER: Testar, implementar
```

### 4.2 Fases

| Fase | Atividades |
|------|-----------|
| **Empatizar** | Pesquisar, observar |
| **Definir** | Sintetizar, criar persona |
| **Idear** | Brainstorm, sketching |
| **Prototipar** | Wireframe, protótipo |
| **Testar** | Testar, iterar |

## 6. UX vs UX Design

| Termo | Significado |
|-------|-----------|
| **UX** | Experiência do usuário (campo) |
| **UX Design** | Processo de criar experiências |
| **UX Designer** | Profissional que cria UX |
| **UX Researcher** | Pesquisa com usuários |
| **UX Writer** | Copy para interfaces |
