# Vocabulary

Skill para expandir vocabulário japonês usando flashcards e repetição espacada.

## Níveis

| Nível | Palavras | Complexidade |
|------|----------|--------------|
| N5 | ~300 palavras | Básica, uso diário |
| N6 | ~500 palavras | Intermediária |
| N3 | ~1000 palavras | Avançada |

## Estrutura do Flashcard

```
[Frente]
Palavra em japonês

[Verso]
Significado em português
Leitura ( hiragana)
Exemplo de frase
```

## Exemplos

### Flashcard 1
```
[Frente] 食べ物

[Verso]
Significado: Comida, alimento
Leitura: たべもの
Exemplo: 食べ物を買います (Comprar comida)
```

### Flashcard 2
```
[Frente] 遊ぶ

[Verso]
Significado: Brincar, sair para brincar
Leitura: あそぶ
Exemplo: 昨日友達と遊びました (Ontem brincou com amigos)
```

## Modo de Prática

1. Apresentar flashcard (frente)
2. Usuário tenta lembrar
3. Revelar verso
4. Usuário indica se acertou ou não
5. Repetir com intervalos

## Repetição Espaçada

| Tentativas | Intervalo |
|------------|------------|
| 1ª erro | Repetir imediatamente |
| 2ª acerto | 10 min |
| 3ª acerto | 1 dia |
| 4ª acerto | 3 dias |
| 5ª acerto | 7 dias |

## Como Usar

1. Escolher tema ou "misturado"
2. O skill apresenta flashcards
3. Você tenta lembrar
4. Recebe feedback
5. Continue até dominar
