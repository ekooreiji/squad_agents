# Prompt 05 - Entrega

Prompt para entregar a documentação.

---

## Objetivo

Entregar a documentação criada com instruções de manutenção.

---

## Instruções

Após revisão, entregar os documentos criados.

### 1. Arquivos Criados

Listar todos os arquivos criados:

| Arquivo | Descrição |
|--------|-----------|
| README.md | Visão geral do projeto |
| INSTALL.md | Guia de instalação |
| CONTRIBUTING.md | Guia de contribuição |
| mkdocs.yml | Configuração mkdocs |

### 2. Estrutura Entregue

```
docs/
├── index.md
├── install.md
├── usage.md
├── CONTRIBUTING.md
├── CHANGELOG.md
└── mkdocs.yml
```

### 3. Informações de Manutenção

Fornecer instruções para manter a documentação:

```
## Manutenção

### Como atualizar

1. Editar arquivos em docs/
2. Executar `mkdocs serve` para testar
3. Commitar mudanças

### Como adicionar nova página

1. Criar arquivo em docs/
2. Adicionar ao nav em mkdocs.yml
3. Testar e commitar
```

---

## Checklist de Entrega

- [ ] Documentação revisada
- [ ] Arquivos organizados
- [ ] Estrutura testada
- [ ] Guia de manutenção entregue
- [ ] Pronto para uso
