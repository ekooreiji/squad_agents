# Illustrator Setup

Configurações e práticas para Illustrator.

---

## Configurações Recomendadas

### 1. Document Setup

```
Tamanho: Custom (variável)
Resolution: 300 dpi (print), 72 dpi (digital)
Color Mode: RGB (digital), CMYK (print)
```

### 2. Estrutura de Camadas

```
📁 [Project Name]
├── 📁 00_Icons
├── 📁 01_Logos
├── 📁 02_Illustrations
├── 📁 03_Assets
└── 📁 04_Exports
```

### 3. Layers Naming

```
[Categoria]_[Nome]
Exemplo: Icons_Arrow_Right
```

---

## Arteboards

### Para Icons

Tamanhos padrão:
- 16x16
- 24x24
- 32x32

### Para Logos

## Icon Grid

### Configuração

```
Grid: 24px
Stroke: 2px
```

### Guidelines

```
- Usar múltiplos de 4px
- Manter visual balance
- Testar em múltiplos sizes
```

## Práticas Recomendadas

1. **Symbols** - Para reusable elements
2. **Groups organize** - Por tipo
3. **Outline text** - Para entrega
4. **Use guides** - Para icon grid
5. **Export clean** - Sem metadados
