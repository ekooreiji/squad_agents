# Introdução ao Component-Driven Development

Metodologia de desenvolvimento criada por Netflix e Google.

---

## O que é CDD

Desenvolvimento baseado em componentes:
- Componentes primeiro
- API clara
- Composição
- Design tokens
- Versionável

---

## Historia

- Netflix (2013)
- Google (Material)
- Today: React, Vue, Angular

---

## Conceitos

1. **Small Components**: Mínimo
2. **Composition**: Combinar
3. **API**: Props, events
4. **Design Tokens**: Cores, fontes

---

## Comparações

| CDD | Traditional |
|-------|-------------|
| Componentes | Pages |
| Props | Global CSS |
| Reutilizável | copy/paste |
| Testável | Silo-ed |

---

## Benefits

- Reutilização
- Manutenção
- Teste
- Documentação
