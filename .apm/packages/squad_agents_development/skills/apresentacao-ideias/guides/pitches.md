# Tipos de Pitch

Guia sobre diferentes tipos de pitch e como criá-los.

---

## 1. Visão Geral de Pitches

| Tipo | Duração | Público | Objetivo |
|------|---------|---------|---------|
| Elevator | 30-60 seg | Qualquer | Interesse |
| Team | 2-5 min | Equipe | Buy-in |
| Client | 5-15 min | Cliente | Aprovação |
| Investor | 10-20 min | Investidor | Investimento |
| Sales | Varía | Comprador | Fechar |

---

## 2. Elevator Pitch

### 2.1 Fórmula

```
[Problema] + [Solução] + [Diferencial]
em 30-60 segundos
```

### 2.2 Template

```
"Somos [Nome], que faz [Resumo].
[nossa solução] para [público].
Diferente de [concorrentes] porque
[diferencial]."
```

### 2.3 Exemplo

```
"Somos AppVendas, que帮你
ecommerce a vender mais.
Nosso checkout one-click
aumentou conversão em 35%."
```

---

## 3. Team Pitch

### 3.1 Objetivo

Obter buy-in da equipe internal.

### 3.2 Estrutura

```
1. Contexto (1 min)
2. Problema (1 min)
3. Solução proposta (2 min)
4. Impacto (1 min)
5. Call to action (1 min)
```

### 3.3 Dicas

- Mostrar benefícios para equipe
- Estar aberto a feedback
- Ser conciso

---

## 4. Client Pitch

### 4.1 Objetivo

Obter aprovação do cliente.

### 4.2 Estrutura

```
1. Entendimento (2 min)
2. Solução (5 min)
3. Proposta (3 min)
4. Perguntas (5 min)
```

### 4.2 Foco

| Pergunta | Resposta |
|---------|----------|
| Quanto custa? |ROI claro |
| Quando fica pronto? | Timeline definida |
| O que preciso fazer? |Minimal |

## 5. Investor Pitch

### 5.1 Estrutura de 10 Slides

| Slide | Conteúdo |
|-------|----------|
| 1 | Título/Logo |
| 2 | Problema |
| 3 | Solução |
| 4 | Tamanho de mercado |
| 5 | Modelo de negocio |
| 6 | Tração |
| 7 | Competição |
| 8 | Equipe |
| 9 | ask |
| 10 | Contacto |

### 5.2 Dicas

- Dados > Histórias
- Foco em ROI
- Equipe forte

---

## 6. Sales Pitch

### 6.1 Objetivo

Fechar venda.

### 6.2 Estrutura

```
1. Relevância (hook)
2. Problema (dor)
3. Solução (proposta)
4. Proof (caso)
5. CTA (próximo passo)
```

### 6.3 Fechar

| Técnica | Quando usar |
|---------|-------------|
| Urgência | Prazo |
| Escassez | Oferta |
| Resumo | Decisão clara |
| Pergunta | Quase fechado |

---

## 7. Quick Reference

### 7.1 Por Duração

| Tempo | Slides | Palavras |
|-------|--------|---------|
| 30 seg | 1 | 50-75 |
| 2 min | 3-5 | 250-300 |
| 5 min | 8-10 | 600-750 |
| 10 min | 12-15 | 1400 |

### 7.2 Estrutura Mínima

```
1. Problema (por que importa)
2. Solução (como resuelve)
3. Por que nós (diferencial)
4. CTA (o que fazer)
```
