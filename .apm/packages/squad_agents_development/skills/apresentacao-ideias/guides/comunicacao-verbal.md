# Comunicação Verbal Eficaz

Guia sobre como comunicar verbalmente com clareza durante apresentações.

---

## 1. Princípios de Comunicação

### 1.1 Regra 10-5-5

| Número | Significado |
|--------|-----------|
| 10 | Palavras por pensée |
| 5 | Segundos por slide |
| 5 | Slides por 10 minutos |

### 1.2 Clarity Principles

| Princípio | Descrição |
|----------|----------|
| Simples | Linguagem do público |
| Conciso | Menos palavras |
| Concreto | Exemplos reais |

---

## 2. Técnicas de Delivery

### 2.1 Opening

```
"Good morning/afternoon"
Pause (2 seg)
"Let's talk about X"
```

### 2.2 Estrutura de Fala

```
[Transição] + [Ideia] + [Suporte]
     ↓           ↓          ↓
  "Além disso" "o problema" "dados mostram"
```

### 2.3 Fechar

```
"That brings us to..."
"Here's what I'm recommending..."
"So what does this mean for you?"
```

---

## 3. Body Language

### 3.1 Postura

| Element | Recomendação |
|---------|--------------|
| Postura | Erguido, aberto |
| Mãos | Gestos naturais |
| Olho | Contacto, não ler |

### 3.2 Movimento

| Situação | Ação |
|----------|------|
| Apresentação | Permaneça no lugar |
| Demo | Aproxime-se da tela |
| ênfase | Movimentos contidos |

---

## 4. Voice Techniques

### 4.1 Pausas

```
IMPORTANTE: Pause after key points
         Pause before answers
```

### 4.2 Variação

| Emoção | Tom |
|--------|-----|
| enthusiasm | Energi upward |
| gravidade | Mais grave |
| Urgência | Mais rápido |

### 4.3 Silencepower

```
Silêncio = Atenção
Não filler words como "uh", "like"
```

---

## 5. Lidando com Perguntas

### 5.1 Estrutura EAR

```
E - Entender a pergunta
A - Confirmar ("Você está perguntando sobre...")
R - Responder (ou "Boa pergunta, vou verificar")
```

### 5.2 Gancho de Tempo

```
"Muito boa pergunta. Vamos
abordar isso agora ou no Q&A?"
```

### 5.3 Deskilling

```
"Não tenho essa informação,
mas posso descobrir e te enviar."
```

---

## 6. Erros Comuns

### 6.1 Evitar

| Erro | Correção |
|------|---------|
| Ler slides | Olhar audiência |
| Monótono | Variar tom |
| Rápido | Praticar pausas |
| "Hum" | Pausas silenciosas |

### 6.2 Checklist Pessoal

- [ ] Sem "likes", "ums", "ahs"
- [ ] Sem apontar para si
- [ ] Sem olhar só para 1 pessoa
- [ ] Pausas nos momentos chaves

---

## 7. Prática

### 7.1 Método

1. Escrever fala completa
2. Ler em voz alta
3. Gravar e ouvir
4. Ajustar
5. Repetir 5x+

### 7.2 Timing

| Duração | Prática mínima |
|---------|---------------|
| 5 min | 30 min |
| 15 min | 2 horas |
| 30 min | 4+ horas |
