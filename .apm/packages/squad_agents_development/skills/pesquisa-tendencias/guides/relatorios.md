Guia sobre como criar relatórios de tendências.

## 1. Estrutura de Relatório

### 1.1 Seções

```
1. Resumo Executivo
2. Tendências Identificadas
3. Análise
4. Recomendações
5. Próximos Passos
```

### 1.2 Resumo Executivo

```
[3-5 linhas]
- O que foi analisado
- Principais descobertas
- Recomendação key
```

## 3. Formato por Tipo

### 3.1 Relatório Executivo

- 1-2 páginas
- Visual minimal
- Insights key

### 3.2 Relatório Detalhado

- 10+ páginas
- Dados completos
- Análises deep

### 3.3 Presentation

- 10-15 slides
- Visuais forte
- Decisões claras

---

## 4. Periodicidade

### 4.1 Tipos

| Tipo | Frequência | Uso |
|------|------------|-----|
| Quick scan | Semanal | Monitorar |
| Report padrão | Mensal | Planning |
| Deep dive | Trimestral | Estratégico |

### 4.2Calendário

| Mês | Entrega |
|-----|---------|
| Janeiro | Trends do ano |
| Abril | Q1 + Q2 |
| Julho | Q3 + Q4 |
| Outubro | Q4 + Q1 próximo |
