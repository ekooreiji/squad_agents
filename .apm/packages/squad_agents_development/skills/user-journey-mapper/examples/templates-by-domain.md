# Templates by Domain

Journey map templates adaptados para diferentes domínios e tipos de produto.

---

## E-commerce

### Estágios Típicos

1. **Descoberta** - Procura, navegação, comparação
2. **Consideração** - Visualização de produto, reviews
3. **Decisão** - Adicionar ao carrinho, checkout, pagamento
4. **Uso** - Recebimento, uso do produto, suporte
5. **Advocacy** - Review, recomendação, recompra

### Touchpoints Específicos

- Busca no site, categoria, filters
- Product detail page (PDP)
- Cart, checkout flow
- Email de confirmação, tracking
- Programa de fidelidade

### Métricas Chave

- Add to cart rate
- Cart abandonment rate
- Conversion rate
- AOV (Average Order Value)
- Repeat purchase rate

## Templates por Objetivo

| Objetivo | Estágios Recomendados |
|----------|----------------------|
| Aumentar conversão | Descoberta → Decisão |
| Melhorar retention | Uso → Advocacy |
| Reduzir churn | Consideração → Decisão |
| Aumentar NPS | Uso completo |
| Expandir uso | Consideração → Decisão → Uso |

## Variaciones por Tipo de Usuário

### Usuário Novo

- Foco em: Primeiro contato, onboarding
- Estágios: Descoberta, Consideração, Decisão
- Métricas: First to second action

### Usuário Existente

- Foco em: Adoption, upgrades
- Estágios: Uso, Advocacy
- Métricas: Feature adoption, NRR

### Usuário Inativo

- Foco em: Re-engajamento
- Estágios: Todos (revisitados)
- Métricas: Reactivation rate
