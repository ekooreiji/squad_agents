# Usability Testing

Guia para conduzir testes de usabilidade.

---

## 1. O que é Teste de Usabilidade

Teste de usabilidade é observação direta de usuários completando tarefas.

---

## 2. Tipos

### 2.1 Moderado vs Não Moderado

| Tipo | Vantagens | Desvantagens |
|------|-----------|--------------|
| **Moderado** | Profundidade, flexibility | Custo |
| **Não moderado** | Escala | Menos contexto |

### 2.2 Remoto vs Presencial

| Tipo | Quando Usar |
|------|------------|
| **Remoto** | Equipes distribuídas |
| **Presencial** | Observação contexto |

---

## 3. Processo

### 3.1 Preparação

```
1. Definir objetivos
2. Selecionar tarefas
3. Criar roteiro
4. Recrutar participantes
5. Setup ambiente
```

### 3.2 Condução

```
├── Abertura (5 min)
│   ├── Introdução
│   └── Consentimento
├── Tarefas (30-45 min)
│   ├── [Tarefa 1]
│   └── [Tarefa 2]
├── Questionário (5 min)
└── Entrevista (10 min)
```

---

## 4. Métricas

| Métrica | Definição |
|--------|-----------|
| **Taxa de sucesso** | % que completa tarefa |
| **Tempo** | Tempo para completar |
| **Erros** | Número de erros |
| **Satisfação** | NPS/SUS pós-teste |

---

## 5. Análise

### 5.1 Matriz de Severidade

| Severidade | Critério |
|-----------|----------|
| **Crítica** | Bloqueia tarefa |
| **Maior** | Dificulta muito |
| **Menor** | Causa frustração |
| **Cosmético** | Não afeta |

### 5.2 Template de Relatório

```markdown
## Teste: [Nome]

### Participantes
| ID | Perfil | Experiência |
|----|-------|--------------|
| P1 | ... | ... |

### Problemas

| # | Problema | Severidade | Tarefa |
|-----|----------|-----------|--------|
| 1 | ... | ... | ... |
```
