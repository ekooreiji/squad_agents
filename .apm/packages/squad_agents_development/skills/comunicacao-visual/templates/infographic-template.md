# Infographic Template

Template de infográfico para comunicação visual de dados e informações.

---

## Estrutura 1: Timeline

```
+--------------------------------------------------+
|            TÍTULO DO INFOGRÁFICO                  |
+--------------------------------------------------+

  [Ponto 1] ------ [Ponto 2] ------ [Ponto 3]
      |              |               |
    Data          Data            Data
    Label        Label          Label
   Descrição    Descrição     Descrição
      ↓            ↓             ↓
```
