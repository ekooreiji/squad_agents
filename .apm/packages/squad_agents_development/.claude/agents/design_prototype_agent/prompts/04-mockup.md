# Prompt 04 - Mockup

Prompt para criar mockups de alta fidelidade.

---

## Objetivo

Transformar wireframes em designs visuais completos de alta fidelidade.

---

## Instruções

Após wireframe validado, criar mockups aplicando:

### 1. Tipografia

Usar typography skill:

```
H1: [Font] [Size] [Weight]
H2: [Font] [Size] [Weight]
Body: [Font] [Size] [Weight]
Caption: [Font] [Size] [Weight]
```

### 2. Cores

Usar color-theory skill:

```
Primary: [Hex]
Secondary: [Hex]
Accent: [Hex]
Background: [Hex]
Text: [Hex]
```

### 3. Espaçamento

Usar comunicação-visual:

```
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
2xl: 48px
```

### 4. Componentes

| Componente | Estados | Variações |
|-----------|---------|----------|
| Button | Default, Hover, Active, Disabled | Primary, Secondary |
| Input | Default, Focus, Error, Disabled | - |
| Card | Default, Hover | - |
| Modal | Open, Closed | - |
| Toast | Success, Error, Warning, Info | - |

### 5. Hierarquia Visual

Usar comunicação-visual para criar Hierarchy:

- Tamanho diferenciado
- Peso visual
- Cores de destaque
- Espaçamento

---

## Ferramenta

Figma - Alta fidelidade:
- Auto Layout
- Components
- Variants
- Styles

---

## Checklist de Mockup

- [ ] Tipografia aplicada
- [ ] Cores aplicadas
- [ ] Espaçamento consistente
- [ ] Componentes criados
- [ ] Estados definidos
- [ ] Biblioteca organizada
- [ ] Responsivo
