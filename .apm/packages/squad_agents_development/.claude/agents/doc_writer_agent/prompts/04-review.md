# Prompt 04 - Revisão

Prompt para revisar e validar documentação.

---

## Objetivo

Revisar a documentação criada para garantir qualidade.

---

## Instruções

Após criar os documentos, você deve realizar uma revisão completa.

### 1. Checklist de Revisão

| Item | Verificação |
|------|-------------|
| Conteúdo | Informação clara e completa |
| Estrutura | Seções bem organizadas |
| Formatação | Estilo consistente |
| Exemplos | Código funciona |
| Links | Todos válidos |
| Português | Sem erros de português |

### 2. Pontos de Verificação

**Título e Descrição:**
- [ ] Título claro
- [ ] Descrição breve

**Conteúdo:**
- [ ] Todas as seções necessárias
- [ ] Exemplos funcionais
- [ ] Informações atualizadas

**Verificação Técnica:**
- [ ] Código nos exemplos funciona
- [ ] Comandos testados
- [ ] Links válidos

**Português:**
- [ ] Sem erros de ortografia
- [ ] Pontuação correta
- [ ] Terminologia consistente

---

## Checklist Final de Revisão

- [ ] Revisãopassada
- [ ] Erros corrigidos
- [ ] Exemplos validados
- [ ] Links conferidos
- [ ] Pronto para entrega

---

## Erros Comuns a Evitar

| Erro | Como Evitar |
|------|-------------|
| Links quebrados | Testar todos os links |
| Código inválido | Testar exemplos |
| Informações faltando | Verificar completeness |
| Inconsistências | Padronizar termos |
