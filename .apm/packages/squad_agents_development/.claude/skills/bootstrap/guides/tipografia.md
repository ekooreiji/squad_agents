# Tipografia Bootstrap

Guia sobre tipografia e textos.

---

## 1. Headings

```html
<h1>h1 - 2.5rem</h1>
<h2>h2 - 2rem</h2>
<h3>h3 - 1.75rem</h3>
```

---

## 2. Classes de Display

```html
<h1 class="display-1">Display 1</h1>
<h1 class="display-2">Display 2</h1>
<h1 class="display-3">Display 3</h1>
<h1 class="display-4">Display 4</h1>
```

---

## 3. Text Utilities

| Classe | Descrição |
|--------|-----------|
| text-start | Alinha esquerda |
| text-center | Alinha centro |
| text-end | Alinha direita |
| text-uppercase | Maiúsculas |
| text-lowercase | Minúsculas |
| text-capitalize | Primeiras maiúsculas |

---

## 4. Cores de Texto

```html
<p class="text-primary">Primary text</p>
<p class="text-secondary">Secondary text</p>
<p class="text-success">Success text</p>
<p class="text-danger">Danger text</p>
<p class="text-warning">Warning text</p>
<p class="text-info">Info text</p>
<p class="text-muted">Muted text</p>
```
