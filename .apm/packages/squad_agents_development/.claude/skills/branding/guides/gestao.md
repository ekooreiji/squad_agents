## 1. O que é Governança

Governança é o sistema de regras que mantém a consistência da marca.

## 4. Auditoria

### 4.1 Quando Auditar

| Frequência | Escopo |
|------------|-------|
| Trimestral | Pontos de contato |
| Anual | Marca completa |
| Mudança | Pós-rebrand |

### 4.2 Checklist

- [ ] Logo usage
- [ ] Cores applied
- [ ] Tom consistente
- [ ] Assets atualizados
- [ ] Implicações corretas

## 5. Evolução

### 5.1 Quando Evoluir

| Sinal | Ação |
|-------|------|
| Outdated | Refresh |
| Expansão | Extender |
| Rebranding | Redesenho |

### 5.2 Tipos de Mudança

| Tipo | Impacto | Exemplo |
|------|--------|---------|
| Evolução | Bajo | Cores |
| Refresh | Médio | Logo |
| Rebrand | Alto | Tudo |
