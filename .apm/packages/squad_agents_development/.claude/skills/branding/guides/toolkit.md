# Brand Toolkit

Guia sobre brand toolkit e aplicabilidade.

---

## 1. O que é Brand Toolkit

Brand toolkit é o conjunto de assets e diretrizes para aplicação da marca.

---

## 2. Componentes

### 2.1 Essentials

```
├── Arquivos
│   ├── Logo (AI, SVG, PNG)
│   ├── Ícones
│   └── Fontes
├── Cores
│   ├── Paleta
│   └── Valores
└── Templates
    ├── Apresentação
    ├── Documento
    └── Social
```

### 2.2 Documentação

- brand guidelines

---

## 3. Arquivos

### 3.1 Logo Files

| Formato | Uso |
|--------|-----|
| AI/EPS | Vetor (impressão) |
| SVG | Vetor (digital) |
| PNG | Raster (fundo transparent) |
| JPG | Raster (fundo branco) |

### 3.2 Tamanhos

```
Logo principal: 500px minimum
Social post: 1200x630
Favicon: 32x32
```

---

## 4. Templates

### 4.1 Presets

| Template | Formato |
|---------|--------|
| Apresentação | PowerPoint, Keynote |
| Email signature | HTML |
| Social media | Figma |
| Documentos | Word, G Docs |

### 4.2 Componentes

```
Templates devem incluir:
- Logo correto
- Cores corretas
- Tipografia válida
- Paleta de cores
```
