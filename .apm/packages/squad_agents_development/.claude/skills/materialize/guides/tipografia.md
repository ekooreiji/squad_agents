# Tipografia - Materialize

Texto e headings Materialize.

---

## Headings

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

---

## Flow Text

```html
<p class="flow-text">Flow text responsivo</p>
```

---

## Text Alignment

```html
<p class="left-align">Esquerda</p>
<p class="right-align">Direita</p>
<p class="center-align">Centro</p>
```

---

## Text Colors

```html
<p class="black">Preto</p>
<p class="white">Branco</p>
<p class="red-text">Vermelho</p>
<p class="blue-text">Azul</p>
```

---

## Text Sizes

```html
<p class="truncate">Texto truncado...</p>
```
