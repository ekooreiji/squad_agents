# Comunicação Visual

Skill especializada em transmitir mensagens de forma clara e impactante através de elementos visuais.

---

## Visão Geral

Esta skill ensina a:

- Criar hierarquia visual efetiva
- Compor elementos de forma clara
- Usar tipografia para comunicar
- Aplicar cores com propósito
- Utilizar ícones e ilustrações
- Construir narrativas visuais
