# Exemplos de Comunicação Visual

Exemplos práticos de comunicação visual clara.

---

## Exemplo 1: Landing Page Hero

### Estrutura

```
+--------------------------------------------------+
| [Logo]                      [Nav Links]          |
+--------------------------------------------------+

              TÍTULO PRINCIPAL
            Subtítulo que explica

        [CTA Button]  [Secondary Button]

     [Screenshot do produto]
+--------------------------------------------------+
```

### Análise

| Elemento | Decisão | Razão |
|---------|---------|------|
| Título | 48px Bold | Largest elemento |
| Subtítulo | 20px Regular | secondary |
| CTA | Cor accent | Foco |
| Imagem | 60% width | Suporte |

## Exemplo 3: Social Media Post

### Instagram

```
+---------------------------+
|                           |
|     [Image: 1080x1080]   |
|         centered          |
|                           |
+---------------------------+
       Title Here
  Caption with context...
  #hashtag #hashtag
```

### Análise

| Elemento | Decisão |
|---------|---------|
| Image | Square 1:1 |
| Title | 22px Bold |
| Caption | 18px |
| Hashtags | Muted |

## Exemplo 4: Email Header

+---------------------------+

|    Main content text    |
|    with details...     |

|                           |
+---------------------------+
```

### Análise

| Elemento | Decisão |
|---------|---------|
| CTA | Button centered |
| Width | 600px max |
| Padding | 40px |

## Exemplo 5: Feature Comparison

### Estrutura

```
+---------------------------+---------------------------+
|     [CHECK] Plan A       |    [STAR] Plan B        |
+---------------------------+---------------------------+
| Title                    | Title                   |
| Price                   | Price                   |
+---------------------------+---------------------------+
| ✓ Feature 1             | ✓ Feature 1             |
| ✓ Feature 2             | ✓ Feature 2             |
| ✗ Feature 3            | ✓ Feature 3             |
|                         |                         |
|    [Choose A]          |    [Choose B] - Best   |
+---------------------------+---------------------------+
```

### Análise

| Elemento | Decisão |
|---------|---------|
| Plan B | Destaque (star/tag) |
| Features | Checkmarks |
| Pricing | Clear comparison |

## Exemplo 6: Error State

### Estrutura

```

|    [Icon] Erro          |
+---------------------------+
|                           |
|   Algo deu errado.       |
|   Explicação breve      |

## Comparativo Final

| Tipo | Foco | Técnica |
|------|-----|---------|
| Hero | Conversion | Hierarchy |
| Dashboard | Data | Numbers |
| Social | Engagement | Visual |
| Email | Action | CTA claro |
| Comparison | Decisão | Contraste |
| Error | Recuperação | Simplicidade |
| Success | Confirmação | Positivismo |
