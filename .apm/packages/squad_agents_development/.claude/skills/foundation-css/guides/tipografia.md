# Tipografia - Foundation CSS

Global Typography e headings.

---

## Global Fonts

Foundation usa por padrão:
- **Primary Font:** "Helvetica Neue", Helvetica, Roboto, Arial, sans-serif
- **Monospace:** "Consolas", "Liberation Mono", Courier, monospace

---

## Headings

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

---

## Subheadings

```html
<h1 class="subheader">Subheading 1</h1>
<h2 class="subheader">Subheading 2</h2>
```

---

## Text

```html
<p>Parágrafo normal</p>
<p class="lead">Lead paragraph</p>
<small>Texto pequeño</small>
```

---

## Text Alignment

```html
<p class="text-left">Esquerda</p>
<p class="text-right">Direita</p>
<p class="text-center">Centro</p>
<p class="text-justify">Justificado</p>
```

---

## Text Colors

```html
<p class="text-primary">Primary</p>
<p class="text-secondary">Secondary</p>
<p class="text-success">Success</p>
<p class="text-warning">Warning</p>
<p class="text-alert">Alert</p>
```

---

## Unordered List

```html
<ul class="no-bullet">
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
```

---

## Inline List

```html
<ul class="inline-list">
    <li><a href="#">Link 1</a></li>
    <li><a href="#">Link 2</a></li>
</ul>
```

---

## Definition List

```html
<dl>
    <dt>Termo</dt>
    <dd>Definição</dd>
</dl>
```

---

## Blockquote

```html>
<blockquote>
    <p>Citação</p>
    <cite>Autor</cite>
</blockquote>
```

---

## Text Utils

```html
<p class="text-lowercase">MAÚSCULAS → minúsculas</p>
<p class="text-uppercase">minúsculas → MAÚSCULAS</p>
<p class="text-capitalize">primeira letra maiúscula</p>

<p class="nowrap">Sem quebra de linha</p>
<p class="truncate">Texto truncado...</p>
