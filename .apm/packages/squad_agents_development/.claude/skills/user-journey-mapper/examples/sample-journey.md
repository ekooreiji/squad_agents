## Persona

| Campo | Conteúdo |
|-------|---------|
| **Perfil** | Profissional urbano, 28 anos |
| **Demográfico** | São Paulo, renda média-alta |

## Estágios da Jornada

### 1. Descoberta

| Campo | Conteúdo |
|-------|---------|
| **Ação** | Procura "delivery comida" no Google |
| **Touchpoint** | Busca orgânica, Instagram ads |
| **Sentimento** | 😐 Neutro - apenas pesquisando |
| **Pensamento** | "Vou ver quais opções tenho" |
| **Pain Point** | Muitos apps similares, qual escolher? |
| **Opportunity** | Ter onboarding rápido e diferenciado |
| **Métricas** | 1000 visits/mês, bounce 60% |

### 2. Consideração

| Campo | Conteúdo |
|-------|---------|
| **Ação** | Abre app, navega por restaurantes |
| **Touchpoint** | App FoodApp |
| **Sentimento** | 😊 Satisfeito - interface limpa |
| **Pensamento** | "Vou pedir do japonês que eu gosto" |
| **Pain Point** | - |
| **Opportunity** | Recommendações personalizadas |
| **Métricas** | Avg time 3 min, 70% find restaurant |

### 3. Decisão

| Campo | Conteúdo |
|-------|---------|
| **Ação** | Seleciona restaurante, coloca itens no carrinho |
| **Touchpoint** | Checkout do app |
| **Sentimento** | 😟 Frustrado - taxa de entrega alta |
| **Pensamento** | "Nossa, R$ 15 de entrega? É muito" |
| **Pain Point** | Taxa de entrega variável e alta |
| **Opportunity** | Mostrar taxa fixa oder waived above threshold |
| **Métricas** | Cart abandonment 45% |

### 4. Uso

| Campo | Conteúdo |
|-------|---------|
| **Ação** | Acompanha status do pedido |
| **Touchpoint** | Push notifications, tracking in-app |
| **Sentimento** | 😁 Muito satisfeito - entrega rápida |
| **Pensamento** | "Chegou em 35 min, rapidão" |
| **Pain Point** | - |
| **Opportunity** | - |
| **Métricas** | Avg delivery 35 min, NPS 72 |

### 5. Advocacy

| Campo | Conteúdo |
|-------|---------|
| **Ação** | Deixa review, indica amigos |
| **Touchpoint** | App store, WhatsApp |
| **Sentimento** | 😁 Muito satisfeito |
| **Pensamento** | "Vou pedir de novo, super recomendo" |
| **Pain Point** | - |
| **Opportunity** | Programa de fidelidade com discounts |
| **Métricas** | NPS 72, 30% referral rate |

---

## Análise Consolidada

### Pain Points Identificados

| Estágio | Pain Point | Severidade | Impacto |
|--------|-----------|-----------|---------|
| Descoberta |Many apps similares | Baixa | Usuário pode escolher outro |
| Decisão | Taxa de entrega alta | Alta | 45% cart abandonment |
| Decisão | Tempo mínimo alto | Média | Usuário去寻找outros |

### Delight Points Identificados

| Estágio | Delight Point | Impacto |
|--------|--------------|---------|
| Consideração | Interface limpa e intuitiva | Alto |
| Uso | Entrega rápida | Muito alto |
| Uso | Tracking em tempo real | Alto |

### Oportunidades Priorizadas

| Opportunity | Estágio | Impacto | Esforço | Prioridade |
|-------------|--------|--------|--------:|----------|
| Taxa fixa ou gratuita > R$50 | Decisão | Alto | Médio | P1 |
| Recommendações personalizadas | Consideração | Médio | Alto | P2 |
| Programa de fidelidade | Advocacy | Alto | Médio | P2 |
