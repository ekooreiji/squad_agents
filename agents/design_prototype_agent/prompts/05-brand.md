# Prompt 05 - Brand Application

Prompt para aplicar marca ao design.

---

## Objetivo

Aplicar identidade de marca aos mockups.

---

## Instruções

Após mockups prontos, aplicar branding:

### 1. Logo

- Position correta
- Size legível
- Clear space

### 2. Cores de Marca

Se existente (usar branding skill):

| Cor de marca | Usar em |
|------------|--------|
| Primary | CTAs, Destaques |
| Secondary | Fundos, secondary |
| Accent | Highlights |

Se não existente:

- Criar paleta baseada em brief
- Justificar escolhas
- Sugerir evolução

### 3. Tipografia de Marca

Se existente:
- Usar fontes definidas

Se não existente:
- Sugerir based no posicionamento
- Justificar escolhas

### 4. Imagery Style

| Estilo | Descrição |
|-------|-----------|
| Fotos | Definir estilo |
| Ilustrações | Definir estilo |
| Ícones | Definir set |

---

## Consistência

Verificar:
- [ ] Logo consistente
- [ ] Cores consistentes
- [ ] Tipografia consistente
- [ ] Espaçamento consistente

---

## Ferramentas

Figma:
- Styles de cores
- Text styles
- Components

---

## Checklist de Brand

- [ ] Logo aplicado
- [ ] Cores de marca aplicadas
- [ ] Tipografia aplicada
- [ ] Consistência verificada
- [ ] Style guide atualizado
