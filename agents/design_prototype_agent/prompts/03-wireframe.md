# Prompt 03 - Wireframe

Prompt para criar wireframes de baixa fidelidade.

---

## Objetivo

Criar estrutura de telas em baixa fidelidade.

---

## Instruções

Após discovery, criar wireframes com:

### 1. Estrutura Base

Cada tela deve ter:

```
[Header]
[Navigation]
[Content Area]
[Sidebar - se aplicável]
[Footer]
```

### 2. Componentes Wireframe

| Componente | Símbolo |
|-----------|---------|
| Imagem | □ |
| Texto | === |
| Botão | ( ) |
| Input | [] |
| Imagem | ○ |

### 3. Layout

- Usar grid básico
- Manter espaços consistentes
- Priorizar funcionalidade sobre estética

### 4. Responsividade

```
 Desktop: 12-col grid
 Tablet: 8-col grid
 Mobile: 4-col grid
```

---

## Ferramenta

Usar Figma para wireframes com:
- Shapes básicas
- Texto simples
- Cinzal gray scale

---

## Checklist de Wireframe

- [ ] Todas as telas mapeadas
- [ ] Fluxos representados
- [ ] Componentes identificados
- [ ] Responsividade considerada
- [ ] Estrutura validada
