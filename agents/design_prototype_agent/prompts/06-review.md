# Prompt 06 - Review

Prompt para revisar e finalizar protótipos.

---

## Objetivo

Revisar design e preparar para entrega.

---

## Instruções

Após brand aplicado, realizar revisão:

### 1. Checklist de Validação

| Item | Verificação |
|------|-------------|
| Design fiel ao brief | [ ] |
| Usuários considerados | [ ] |
| Fluxos funcionais | [ ] |
| Consistência visual | [ ] |
| Marca aplicada | [ ] |
| Responsividade | [ ] |
| Acessibilidade | [ ] |
| Hierarquia clara | [ ] |
| Componentes funcionais | [ ] |

### 2. Teste com Usuários

Se possível:
- Coletar feedback
- Identificar Issues
- Iterar design

Realizar ajustes necessários:

```

1. [Ajuste 1]
2. [Ajuste 2]
```
