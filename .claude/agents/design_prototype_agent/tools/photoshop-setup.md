# Photoshop Setup

Configurações e práticas para Photoshop.

---

## Configurações Recomendadas

### 1. Document Setup

```
Tamanho padrão: 1920x1080
Resolution: 72 dpi
Color Mode: RGB 8-bit
```

### 2. Estrutura de Camadas

```
📁 [Project Name]
├── 📁 00_Wireframes
├── 📁 01_Mockups
├── 📁 02_Assets
├── 📁 03_Images
└── 📁 04_Exports
```

### 3. Camadas Naming

```
[Componente]_[Estado]_[Versão]
Exemplo: Button_Primary_Hover_v01
```
