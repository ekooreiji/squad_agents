# Prompt 02 - Discovery

Prompt para fase de pesquisa e descoberta.

---

## Objetivo

Pesquisar usuários, fluxos e requisitos visuais.

---

## Instruções

Após kickoff, você deve realizar pesquisa para entender:

### 1. Personas

Usar UX Design skill para criar personas:

```
## Persona 1
### Informações
- Nome:
- Idade:
- Profissão:

### Objetivos
- [Goal 1]
- [Goal 2]

### Frustrations
- [Pain 1]
- [Pain 2]

### Comportamento
- [Comportamento]
```

### 2. User Flows

Mapear jornadas:

```
## Fluxo 1: [Nome do fluxo]
1. [Step]
2. [Step]
3. [Step]
```

### 3. Requisitos Visuais

Extrair do brief:

## Análise Competitiva
