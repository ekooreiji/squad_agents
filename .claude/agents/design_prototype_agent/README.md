# design_prototype_agent

Agente especializado em prototipação visual de telas.

---

## Sobre

O **design_prototype_agent** cria protótipos visuais de alta fidelidade usando ferramentas de design como Figma, Photoshop e Illustrator. O agente não desenvolve código, apenas cria a parte visual.

---

## Ferramentas

| Ferramenta | Prioridade |
|-----------|------------|
| Figma | Alta ✓ |
| Photoshop | Média |
| Illustrator | Baixa |

---

## Fluxo de Trabalho

```
Kickoff → Discovery → Wireframe → Mockup → Brand → Review
```

---

## Como Usar

1. **Enviar brief** - Descrever o projeto
2. **Agente pergunta** - Para esclarecer requisitos
3. **Agente cria** - Wireframes, mockups, style guide
4. **Revisão** - Ajustes necessários
5. **Entrega** - Arquivos prontos para development
