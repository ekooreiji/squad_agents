# Prompt 03 - Formatação

Prompt para aplicar formatação aos documentos.

---

## Objetivo

Aplicar formatação correta aos documentos em português.

---

## Instruções

Ao escrever o conteúdo, você deve aplicar a formatação padrão.

### 1. Elementos de Formatação

| Elemento | Sintaxe | Exemplo |
|---------|--------|---------|
| Título nível 1 | # | # Título |
| Título nível 2 | ## | ## Subtítulo |
| Título nível 3 | ### | ### Seção |
| Negrito | ** | **texto** |
| Itálico | * | *texto* |
| Código inline | ` | `código` |
| Link | [texto](url) | [docs](https://...) |
| Imagem | ![alt](url) | ![logo](img.png) |

### 2. Blocos de Código

````markdown
```python
def hello_world():
    print("Olá, mundo!")
```
````

### 3. Listas

**Lista não ordenada:**
- Item 1
- Item 2
- Item 3

**Lista ordenada:**
1. Primeiro item
2. Segundo item
3. Terceiro item

**Lista de tarefas:**
- [ ] Tarefa pendente
- [x] Tarefa concluída

### 4. Tabelas

| Coluna 1 | Coluna 2 | Coluna 3 |
|---------|----------|----------|
| Dado 1 | Dado 2 | Dado 3 |
| Dado 4 | Dado 5 | Dado 6 |

### 5. Blocos de Aviso

````markdown
!!! note
    Este é uma nota informativa.

!!! warning
    Este é um aviso importante.

!!! tip
    Este é uma dica útil.
````
