# Introdução ao Design Thinking

Guia introdutório sobre Design Thinking.

---

## 1. O que é Design Thinking

Design Thinking é uma abordagem centrado no usuário para resolver problemas complexos.

---

## 2. Origem

### 2.1 Stanford d.school

- Criado na Stanford University
- Professors: David Kelley, IDEO
- Baseado em designer's ways of thinking

### 2.2 Evolução

| Período | Desenvolvimento |
|--------|---------------|
| 1960s | Design methods movement |
| 1980s | Human-centered design |
| 1990s | Design thinking term |
| 2000s | d.school, popularização |

---

## 3. Os 5 Estágios do DT

### 3.1 Modelo IDEO

```
1. EMPATIA     → Entender usuários
2. DEFINIR    → Problem statement
3. IDEAR      → Brainstorming
4. PROTOTIPAR → Criar protótipos
5. TESTAR    → Validar com usuários
```

### 3.2 Modelo Stanford

```
    EMPATIA
        ↓
    DEFINIR ←→ IDEAR ← PROTOTIPAR
        ↓           ↓
      TESTAR ←------+
```

## 4. Princípios Fundamentais

### 4.1 Centrado no Humano

```
"Sempre entender quem está nocentro do problema"
```

### 4.2 Iterativo

```
"Não é processo linear, é cíclico"
```

### 4.3 Experimental

```
"Prototipar para aprender"
```

### 4.4 Colaborativo

```
"Múltiplas perspectivas = melhores soluções"
```

---

## 5. Por que usar Design Thinking

### 5.1 Benefícios

- Soluções mais inovadoras
- Maior aceitação usuários
- Redução de riscos
- Times mais criativos

### 5.2 Quando Aplicar

| Situação | Adequação |
|----------|-----------|
| Problemas complexos | ✓✓✓ |
| Incerteza | ✓✓✓ |
| Necessidade de inovação | ✓✓✓ |
| Mudança cultural | ✓✓✓ |
