## 2. Ferramentas de Definição

### 2.1 Point of View (POV)

```
Usuário [tipo] precisa [necessidade]
porque [insight]
```

### 2.2 How Might We (HMW)

```
Como podríamos [resolver problema]
de forma que [benefício]?
```

### 2.3 Personas

| Atributo | Descrição |
|---------|-----------|
| Nome | Identificação |
| Demográfico | idade, local |
| Comportamento | Usos, preferências |
| Objetivos | O que quer alcançar |
| Dores | Problemas enfrentados |

## 4. Ferramentas de Prototipagem

### 4.1 Tipos de Protótipo

| Tipo | Fidelidade | Uso |
|------|-----------|-----|
| Papel | Baixa | Testar conceito |
| Wireframe | Média | Estrutura |
| MVP | Alta | Produto real |

### 4.2 Rapid Prototyping

- Materiais simples
- Tempo limitado
- Focus em aprender

## 5. Ferramentas de Testagem

### 5.1 Testes com Usuários

| tipo | Descrição |
|------|-----------|
| Guerrilla | Público aleatório |
| Lab | Ambiente controlado |
| Field | No contexto real |
| Remote | Online |

### 5.2 Métricas

| Tipo | O que medir |
|------|------------|
| Quantitativo | Tempo, taxa, score |
| Qualitativo | Opiniões, sentimentos |
| Behavioral | Ações, navegação |

## 6. Matrizes Úteis

### 6.1 Esforço vs Impacto

```
↑ Impacto
│ ★ ★ ★  │ ● ● ●
│ ★ ★    │ ● ●
│ ★      │ ●
├────────┼──────────→ Esforço
```

### 6.2 Matriz de Decisão

| | Baixo custo | Alto custo |
|---|---|---|
| Rápido | Quick wins | Considerar |
| Lento | Depriorizar | Big bets |
