# One-Pager Template

Template de one-pager para apresentação rápida.

---

## Formato: Uma Página

```
+------------------------------------------+
|              HEADER                      |
|   Logo | Tagline | Contacto            |
+------------------------------------------+
|                                          |
|   O PROBLEMA                            |
|   [1-2 linhas sobre o problema]        |
|                                          |
+------------------------------------------+
|                                          |
|   A SOLUÇÃO                             |
|   [1-2 linhas sobre a solução]         |
|                                          |
+------------------------------------------+
|                                          |
|   BENEFÍCIOS                            |
|   - [Benefício 1]                       |
|   - [Benefício 2]                       |
|   - [Benefício 3]                       |
|                                          |
+------------------------------------------+
|                                          |
|   COMO FUNCIONA                         |
|   [Brief explanation]                    |
|                                          |
+------------------------------------------+
|                                          |
|   PROVA                                 |
|   [Dado/Testimonial]                     |
|                                          |
+------------------------------------------+
|                                          |
|   CTA: [Próximo passo]                   |
|                                          |
+------------------------------------------+
```

---

## Para Projetos

```
+------------------------------------------+
|   [Nome do Projeto]                      |
|   [Data] | [Versão]                   |
+------------------------------------------+
|                                          |
|   RESUMO                                |
|   [2-3 linhas sobre o projeto]           |
|                                          |
+------------------------------------------+
|                                          |
|   ESCOPO                                |
|   Incluído:                             |
|   - [Item]                             |
|   - [Item]                             |
|                                          |
|   Excluído:                             |
|   - [Item]                             |
|                                          |
+------------------------------------------+
|                                          |
|   CRONOGRAMA                            |
|   [Datas principais]                     |
|                                          |
+------------------------------------------+
|                                          |
|   ORÇAMENTO                           |
|   R$ [Valor]                          |
|                                          |
+------------------------------------------+
|                                          |
|   PRÓXIMOS PASSOS                      |
|   [1. Ação] [2. Ação] [3. Ação]        |
|                                          |
+------------------------------------------+
```

---

## Guidelines

| Medida | Valor |
|--------|-------|
| Largura max | 800px |
| Altura max | 1100px |
| Fontes | Máximo 2 |
| Cores | Máximo 3 |
