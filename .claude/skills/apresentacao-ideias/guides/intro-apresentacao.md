# Introdução à Apresentação de Ideias

Guia introdutório sobre apresentação de ideias.

---

## 1. O que é Apresentação de Ideias

Apresentação de ideias é a habilidade de comunicar conceitos de forma clara e persuasiva.

---

## 2. Por que Importa

### 2.1Importância no Design

| Situação | Por que |
|----------|---------|
| Vender projeto | Client precisa entender valor |
| Justificar decisões | Equipe precisa buy-in |
| Obter aprovação | Stakeholders precisam crer |
| Motivar equipe | Alignment necessário |

### 2.2Impacto

- Bons presenters avanzammais
- Ideias são aceitas
- Carreira progride

---

## 3. Tipos de Apresentação

### 3.1 Por Contexto

| Tipo | Duração | Público |
|------|---------|---------|
| Elevator pitch | 30-60 seg | Qualquer |
| Pitch deck | 5-20 min | Investidores |
| Presentation | 20-60 min | Equipe/Cliente |
| TED-style | 10-30 min | Audiência |

### 3.2 Por Objetivo

| Objetivo | Tipo |
|---------|------|
| Vender ideia | Pitch |
| Treinar | Workshop |
| Atualizar | Status report |
| Decidir | Recommendation |

---

## 4. Componentes de uma Boa Apresentação

### 4.1 Conteúdo

| Componente | Descrição |
|------------|-----------|
| Problema |O que resolve |
| Solução | Como funciona |
| Benefício | Por que importa |
| Prova | Evidências |
| CTA | Próximo passo |

### 4.2 Entrega

| Componente | Descrição |
|------------|-----------|
| Clareza | Linguagem simples |
| Estrutura | Fluxo lógico |
| Evidências | Dados, exemplos |
| Paixão | Engajamento |

---

## 5. Processo de Apresentação

### 5.1 Passo a Passo

1. **Definir objetivo** - O que quero que acontece?
2. **Conhecer audiência** - Quem vai ouvir?
3. **Estruturar** - Qual história?
4. **Criar visuales** - Slides, demos
5. **Rehearsar** - Praticar muito
6. **Apresentar** - Executar
7. **Follow-up** - Manter对话

### 5.2 Tempo Típico

| Fase | Tempo |
|------|-------|
| Prep | 80% do esforço |
| Apresentação | 10% |
| Follow-up | 10% |
