# Writing

Skill para praticar a escrita japonesa: hiragana, katakana e kanji.

## O que Praticar

| Sistema | Uso |
|---------|-----|
| Hiragana | Palavras japonesas nativas |
| Katakana | Palavras estrangeiras, nomes |
| Kanji | Palavras com caracteres chineses |

## Modos de Prática

### 1. Dictado
O skill dita uma palavra, usuário escreve.

### 2. Tradução
O skill dá um significado, usuário escreve em japonês.

### 3. Composição
O skill dá um tema, usuário escreve uma frase.

### 4. Correção
Usuário escreve, skill corrige.

## Estrutura dos Exercícios

### Ditado
```
Skill: "ame" (chuva)
Usuário: あめ

✓ Correto!
```

### Tradução
```
Skill:"Escreva 'estudante' em japonês"
Usuário: がくせい

✓ Correto! (学生)
```

### Composição
```
Skill: "Escreva uma frase sobre seu dia"
Usuário: 今日は学校に行きました。

✓ Correto!
```

## Progressão

| Fase | Conteúdo |
|------|---------|
| 1 | Hiragana básico |
| 2 | Hiragana + conectores |
| 3 | Katakana básico |
| 4 | Katakana nomes |
| 5 | Kanji N5 |
| 6 | Kanji N4 |
| 7 | Misturado |

## Pontos de Atenção

### Hiragana
- う (u) não soa como "u" em algumas palavras
- ん (n) é diferente de う (u)
- Duplas (っ) indicam consoante ganda

### Katakana
- Longas vogais (ダッシュ: dāshu)
- Combinações (チャー: chā)

### Kanji
- Ler por kun'yomi ou on'yomi conforme contexto
- Radicals ajudam a lembrar

## Como Usar

1. Escolher modo (ditado/tradução/composição)
2. Escolher nível (hiragana/katakana/kanji/misturado)
3. PraticarWriting
4. Receber correção

## Desafios

### 1. Frases Simples
- 私は学生です (Sou estudante)
- 昨日友達と公園に行きました (Ontem fui ao parque com amigos)

### 2. Diálogo
- すみません、いくらですか? (Com licença, quanto custa?)
- お願いします (Por favor)

### 3. Composição Livre
- Escrever sobre: família, rotina, hobbies, viagem

## Boas Práticas

- Escrever à mão diariamente
- Não usar teclado
- Revisar erros frequentemente
- Praticar kanji com palavras conhecidas
