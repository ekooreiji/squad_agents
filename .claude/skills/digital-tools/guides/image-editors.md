## 3. Guia de Uso

### 3.1 Fluxo de Edição

```
1. Importar imagem
   │
2. → Avaliar → Correções básico
   │
3. → Adjust colors → Levels, Curves
   │
4. → Retouch → Clone, Healing
   │
5. → Composite → Layers, Masks
   │
6. → Output → Export
```

### 3.2 Common Tasks

| Tarefa | Ferramenta | Settings |
|--------|-------------|----------|
| Color correction | Curves, Levels | RGB, midtones |
| Remove object | Content-Aware Fill | Clone/Healing brush |
| Combine images | Layer masks | Soft edge, 10px |
| Resize | Image > Image size | Constrain proportions |
