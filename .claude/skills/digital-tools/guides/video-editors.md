# Video Editors

Guia completo de ferramentas de edição de vídeo.

---

## 1. Ferramentas Profissionais

### 1.1 Adobe Premiere Pro

| Recurso | Descrição |
|---------|-----------|
| melhor para | Professional editing, integração CC |
| Plataforma | macOS, Windows |
| Custo | Subscription |

**Atalhos Essenciais:**

| Atalho | Função |
|--------|--------|
| J/K/L | Rewind/Forward/Pause |
| Space | Play/Pause |
| C | Razor tool |
| V | Selection tool |
| Cmd/Ctrl+K | Add cut |
| Cmd/Ctrl+Shift+K | Ripple delete |

### 1.2 DaVinci Resolve

| Recurso | Descrição |
|---------|-----------|
| melhor para | Color grading, gratuito |
| Versão Gratuita | Resolve (funcionalidades básicas) |
| Curva de aprendizado | Média-Alta |

## 3. Fluxo de Edição

### 3.1 Estrutura de Projeto

```
Project → Bins → Sequences → Clips
```

### 3.2 Fluxo de Trabalho

```
1. Setup
   │ New project
   │ Import footage
   │
2. Organize
   │ Create bins
   │ Label clips
   │
3.rough Cut
   │ Assemble clips
   │ Set timing
   │
4. Fine Cut
   │ Refine transitions
   │ Add effects
   │
5. Color
   │ Color correction
   │ Grading
   │
6. Audio
   │ Mix levels
   │ Add music
   │
7. Export
   │ Settings
   └── Render
```
