# Digital Tools Skill

Skill especializada em dominar ferramentas digitais para design, edição de vídeo, prototipagem e pesquisa.

---

## Visão Geral

Esta skill cobre todo o espectro de ferramentas digitais: de editors de imagem a ferramentas de prototipagem e pesquisa.
