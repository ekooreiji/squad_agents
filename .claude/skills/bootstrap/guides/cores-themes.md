# Cores e Temas Bootstrap

Guia sobre cores e temas.

---

## 1. Cores Padrão

| Nome | Classe bg | Classe text | Classe border |
|------|-----------|------------|--------------|
| Primary | bg-primary | text-primary | border-primary |
| Secondary | bg-secondary | text-secondary | border-secondary |
| Success | bg-success | text-success | border-success |
| Danger | bg-danger | text-danger | border-danger |
| Warning | bg-warning | text-warning | border-warning |
| Info | bg-info | text-info | border-info |
| Light | bg-light | text-light | border-light |
| Dark | bg-dark | text-dark | border-dark |

---

## 2. Background Colors

```html
<div class="bg-primary">Primary</div>
<div class="bg-secondary">Secondary</div>
<div class="bg-success">Success</div>
```

---

## 3. Gradients (BS5)

```html
<div class="bg-gradient">Com gradiente</div>
```
