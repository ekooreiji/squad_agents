# Bootstrap Skill

Skill especializada em Bootstrap para criar interfaces modernas e responsivas.

---

## Visão Geral

Esta skill ensina a:

- Configurar e usar Bootstrap
- Implementar sistema de grid
- Criar componentes reuseis
- Desenvolver formulários
- Aplicar tipografia
- Customizar temas
- Criar designs responsivos
