# Security Basics

## Princípios Fundamentais

| Princípio | Descrição |
|----------|-----------|
| **CIA** | Confidencialidade, Integridade, Disponibilidade |
| **Defense in Depth** | Múltiplas camadas de segurança |
| **Least Privilege** | Mínimo privilégio necessário |
| **Fail Secure** | Falhas devem ser seguras |
| **Don't Trust** | Não confiar em entradas |

## Termos

| Termo | Significado |
|-------|-----------|
| **Vulnerabilidade** | Fraqueza explorável |
| **Ameaça** | Potencial exploração |
| **Exploit** | Uso de vulnerabilidade |
| **Mitigação** | Proteção contra ameaça |

## Categorias de Segurança

- Application Security
- Data Security
- Network Security
- Infrastructure Security
