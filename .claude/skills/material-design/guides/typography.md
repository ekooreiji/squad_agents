# Tipografia - Material Design

Sistema de tipografia.

---

## Font Family

Roboto (default)

---

## Type Scale

| Style | Font | Size | Weight | Line |
|-------|------|------|--------|------|
| H1 | Roboto | 96sp | Light | 112sp |
| H2 | Roboto | 60sp | Light | 72sp |
| H3 | Roboto | 48sp | Regular | 56sp |
| H4 | Roboto | 34sp | Regular | 48sp |
| H5 | Roboto | 24sp | Regular | 32sp |
| H6 | Roboto | 20sp | Medium | 24sp |
| Subtitle | Roboto | 16sp | Regular | 24sp |
| Body1 | Roboto | 16sp | Regular | 24sp |
| Body2 | Roboto | 14sp | Regular | 20sp |
| Button | Roboto | 14sp | Medium | 16sp |
| Caption | Roboto | 12sp | Regular | 16sp |
| Overline | Roboto | 10sp | Regular | 16sp |

---

## Hierarchy

- H1-H6: Headlines
- Body1, Body2: Body text
- Caption: Captions
- Button: Buttons

---

## Usage

```css
:root {
  --md-font-family: 'Roboto', sans-serif;
}

h1 { font: 300 96sp/112sp Roboto; }
p { font: 400 16sp/24sp Roboto; }
button { font: 500 14sp/16sp Roboto; text-transform: uppercase; }
```
