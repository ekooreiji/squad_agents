## Color Roles

- **Primary**: Main actions
- **Secondary**: FAB, highlights
- **Background**: Page surfaces
- **Error**: Error states

---

## Usage

```css
:root {
  --md-primary: #2196F3;
  --md-secondary: #FF5722;
  --md-error: #F44336;
}
```
