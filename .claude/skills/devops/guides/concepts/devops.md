# O que é DevOps

## Definição

DevOps é a prática que combina desenvolvimento (Dev) e operações (Ops) para entregar software de forma mais rápida e confiável.

## Ciclo DevOps

```
┌─────────────────────────────────────────────────────────────┐
│                  Ciclo DevOps                                │
├─────────────────────────────────────────────────────────────┤
│                                                          │
│    Plan → Code → Build → Test → Deploy → Operate → Monitor   │
│            ↑                                ↓               │
│            └────────────────────────────────┘              │
│                                                          │
└─────────────────────────────────────────────────────────────┘
```

## Fases

| Fase | Descrição |
|------|-----------|
| **Plan** | Planejar funcionalidades |
| **Code** | Escrever código |
| **Build** | Build e compile |
| **Test** | Testes automatizados |
| **Deploy** | Deploy para staging/prod |
| **Operate** | Operar a aplicação |
| **Monitor** | Monitorar e feedback |

## Benefits

| Benefício | Descrição |
|----------|-----------|
| **Velocidade** | Deploys mais frequentes |
| **Confiabilidade** | Mais estáveis |
| **Escala** | Maior capacidade |
| **Colaboração** | Time melhor integrado |
