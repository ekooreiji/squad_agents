# Responsivo - Materialize

Design responsivo com Materialize.

---

## Breakpoints

| Breakpoint | Largura |
|------------|---------|
| s | < 600px |
| m | ≥ 600px |
| l | ≥ 992px |
| xl | ≥ 1200px |

---

## Responsive Classes

```html
<!-- Hide/show -->
<div class="hide-on-small-only">Hide small</div>
<div class="hide-on-med-and-down">Hide med</div>
<div class="hide-on-large-only">Hide large</div>

<!-- Show -->
<div class="show-on-small">Show small</div>
<div class="show-on-medium">Show medium</div>
```

---

## Responsive Grid

```html
<div class="row">
    <div class="col s12 m6 l4">Responsivo</div>
```
