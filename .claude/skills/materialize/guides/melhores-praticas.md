# Boas Práticas - Materialize

Boas práticas ao usar Materialize.

---

## Estrutura

- Usar semantic markup
- Container > Row > Col
- Manter hierarquia

---

## Grid

- Máximo 12 colunas
- Breakpoints: s, m, l, xl

---

## jQuery

- Carregar jQuery antes do Materialize
- M.AutoInit() para inicializar

---

## Performance

- Minificar CSS/JS
- CDN para produção

---

## Acessibilidade

- Labels associados
- semantic markup

---

## Cross-browser

- Testar Chrome, Firefox, Safari
