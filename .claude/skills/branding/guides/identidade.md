# Identidade de Marca

Guia sobre identidade visual da marca.

---

## 1. O que é Identidade

Identidade é a expressão visual da marca - tudo que pode ser visto.

---

## 2. Elementos de Identidade

### 2.1 Core

| Elemento | Descrição |
|-----------|-----------|
| Logo | Símbolo principal |
| Logotipo | Nome em fonte |
| Symbol | Ícone isolated |
| Signature | Combinação |

### 2.2 Expandida

| Elemento | Uso |
|----------|-----|
| Cores | Paleta definida |
| Tipografia | Fontes da marca |
| Fotografia | Estilo de imagens |
| Ilustração | Estilo gráficos |

---

## 3. Logo

### 3.1 Tipos

| Tipo | Exemplo | Quando |
|------|---------|--------|
| Wordmark | Google | Nome único |
| Lettermark | IBM | Abreviação |
| Pictorial | Apple | Visual iconic |
| Abstract | Nike | Abstrato |
| Mascot | Kool-Aid | Personagem |
| Emblem | Starbucks | Badge |

### 3.2 Guidelines

```
- Clear em todos os tamanhos
- Works em negativo
- Flexible para anim
- Distinctive e memorable
```

---

## 3.3 Logo System

```
Primary Logo
├── Full color
├── Single color
└── Reversive

Alternate Mark
├── Icon only
└── Wordmark alone
```

## 4. Cores

### 4.1 Paleta

| Cor | Descrição | Uso |
|-----|-----------|-----|
| Primary | Main | Principal |
| Secondary | Support | Secundário |
| Accent | Highlight | CTAs |
| Neutral | Background | Fundos |

### 4.2 Aplicações

| Uso | Cores permitidas |
|-----|---------------|
| Background | White, brand dark |
| Text | Brand dark, white |
| Buttons | Primary, accent |
| Links | Accent |
