# Exemplos de Branding

Exemplos práticos de branding.

---

## Exemplo 1: Tech Startup

### Brief

```
Missão: Democratizar education
Visão: Education accessible para todos
Valores: Acessibilidade, inovação, impacto
```

### Posicionamento

```
Para students que precisam,
TechLearn é education platform
que democratiza learning.

Different de traditional schools,
TechLearn oferece flexible learning.
```

### Resultado

- Logo: Lettermark com cores vibrantes
- Cores: Azul (trust) + Amarelo (energia)
- Voz: Friendly, direta

---

## Exemplo 2: Restaurant

### Brief

```
Missão: Food de quality accessible
Visão: Best eating experience
Valores: Quality, hospitality, sustainability
```

### Posicionamento

```
Para food lovers que valorizam,
Gourmet é restaurant
que oferece quality sem premium.

Different de fancy restaurants,
Gourmet delivers comfort food.
```

### Resultado

- Logo: Symbolic (garfo + faca)
- Cores: Verde (fresh) + Black
- Voz: Warm, welcoming

---

## Exemplo 3:Tech App

### Brief

```
Missão: Simplificar finance
Visão: Finance para todos
Valores: Simplicity, transparency, trust
```

### Posicionamento

```
Para people que precisam de bank,
Nubank é fintech
que simplify banking.

Different de traditional banks,
Nubank oferece digital experience.
```

### Resultado

- Logo: Roxo + 8 circles
- Voz: Direct, friendly
- messaging: "Bank da pessoas"
