# Hierarquia Visual

Guia sobre como criar hierarquia visual para guiar o olhar.

---

## 1. O que é Hierarquia Visual

Hierarquia visual é a organização de elementos que guia o olhar do espectador na ordem desejada.

---

## 2. Princípios de Hierarquia

### 2.1 Tamanho

```
Título grande → Primeiro olhar
Corpo pequeno → Depois
```

| Elemento | Proporção |
|----------|----------|
| Título | 200-300% do corpo |
| Subtítulo | 150% do corpo |
| Corpo | Base (100%) |
| Caption | 75% do corpo |

### 2.2 Peso Visual

```
Negrito = Mais atenção
Regular = Menos atenção
```

| Peso | Uso |
|------|-----|
| Bold | Mensagem principal |
| Semi-bold | Subtítulos |
| Regular | Corpo |
| Light | Secondary info |

### 2.3 Cor

```
Cor vívida = Primeiro olhar
Cor pastel = Depois
```

| Intensidade | Uso |
|-------------|-----|
| Viva/saturada | Primary message |
| Média | Secondary |
| Pastel/cinza | Background info |

## 3. Técnicas de Hierarchy

### 3.1 Rule of One

```
✓UMA ideia principal por seção
✗ Múltiplas ideias competindo
```

### 3.2 Scanning Patterns

| Padrão | Cultura | Uso |
|--------|---------|-----|
| F | Ocidental | Lefty-done |
| Z | Geral | Top-left → right → down |
| Layer | Digital | Layer cake |

### 3.3 Zonas Quentes

```
+------------------------+
|  UL |    UR          | ← Alta atenção
|     +---------------+
|     |              |
| LL  +---------------+  LR  ← Média atenção
+------------------------+
```

---

## 4. Criando Hierarchy

### 4.1 Tipografia

```
✓ TAMANHO + PESO + COR
   ↓         ↓        ↓
  "OFERTA"  BOLD    VERMELHO
```

### 4.2 Espaçamento

```
✓ Espaço = Atenção

✓✓✓    ← Itens próximos = relacionados
   ↔↔↔    ← Itens distantes = separados


```

### 4.3 Contraste

```
✓ Contraste = Destaque

Mais escuro/l claro = Primeiro olhar
Mesmo valor = Mesma importância
```

## 5. Exemplos Práticos

### 5.1 Hero Banner

```
[=========================]
|                         |
|      TÍTULO PRINCIAL   | ← Grande + Bold + Cor
|      Subtítulo aqui    | ← Médio + Regular
|                         |
|     [BOTÃO]    [CTA]  | ← Secondary
|                         |
[=========================]
```

### 5.2 Card

```
+------------------+
| [IMAGEM]        |
+------------------+
| TÍTULO         | ← 18px Bold
| Subtítulo      | ← 14px Medium
| Texto...      | ← 14px Regular
+------------------+
```

### 5.3 Apresentação Slide

```
=======================
     TÍTULO (40px)
=======================
     Subtítulo (24px)
=======================
      Bullet dots
        - Item 1
        - Item 2
=======================
```

## 6. Testing Hierarchy

### 6.1 Teste de 5 Segundos

1. Mostrar por 5 segundos
2. Perguntar: "O que você viu primeiro?"
3. Se não for a mensagem principal → Falhou

### 6.2 Teste deBlur

1. Aplicar blur na imagem
2. O que permanece legível = Mais importante
