# Introdução à Comunicação Visual

Guia introdutório sobre comunicação visual.

---

## 1. O que é Comunicação Visual

Comunicação visual é a capacidade de transmitir ideias e mensagens através de elementos visuais.

---

## 2. Por que Importa

### 2.1 Stats

| Fato | Significance |
|------|--------------|
| 90% da informação é visual | processamento mais rápido |
| 60x mais rápido | vs texto |
| 6x mais engajamento | com imagens |

### 2.2 Benefícios

- Clareza imediata
- Memória visual
- Engajamento
- Acessibilidade

---

## 3. Princípios Fundamentais

### 3.1 Clareza

A mensagem deve ser understood em segundos.

```
✓ Bom: Ícone + texto curto
✗ Ruim: Texto denso
```

### 3.2 Concisão

Menos é mais.

```
✓ 1 mensagem forte
✗ 5 mensagens fracas
```

### 3.3 Contexto

Considerar onde será visto.

| Contexto | Abordagem |
|----------|----------|
| Mobile | Simples, touch-friendly |
| Print | Detalhado |
| Apresentação | Passo a passo |
| Social | Break-scroller |

---

## 4. Elementos da Comunicação Visual

### 4.1 Hierarchy

| Nível | Elemento | Uso |
|-------|---------|-----|
| 1 | Título principal | Attenção first |
| 2 | Subtítulo | Contexto |
| 3 | Corpo | Detalhes |
| 4 | Support | Secondary info |

### 4.2 Cores

| Função | Uso |
|--------|-----|
| Primária | Main message |
| Secondary | Support |
| Accent | CTA |

### 4.3 Espaço

```
✓ Espaço branco = respiração
✗ Espaço denso = confusão
```

---

## 5. Processo de Criação

### 5.1 Passo a Passo

1. **Definir mensagem** - O que comunicar?
2. **Identificar público** - Quem vai ver?
3. **Escolher canais** - Onde será exibido?
4. **Criar hierarchy** - Qual ordem?
5. **Aplicar elementos** - Design
6. **Testar** - Validar compreensão

### 5.2 Checklist Inicial

- [ ] Mensagem definida?
- [ ] Público identificado?
- [ ] Canais definidos?
- [ ] Tempo de exposição?
