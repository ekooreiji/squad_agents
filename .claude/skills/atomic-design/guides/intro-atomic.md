# Introdução ao Atomic Design

Metodologia de design de interfaces criada por Brad Frost.

---

## O que é Atomic Design

Metodologia que usa química como metáfora:
- **Átomos** = elementos básicos
- **Moléculas** = combinação simples
- **Organismos** = componentes complexos
- **Templates** = layouts
- **Páginas** = instâncias completas

---

## Conceito

Não é um processo linear, mas uma mentalidade de construção.

---

## Benefícios

- Componentes reutilizáveis
- Consistência visual
- Manutenção facilitada
- Documentação clara

---

## Diferenças vs Tradicional

| Tradicional | Atomic Design |
|------------|------------|
| Pages first | Componentes first |
| copy/paste | Reutilização |
| Silo-ed | Modular |
