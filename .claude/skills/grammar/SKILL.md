# Grammar

Skill para prática de gramática japonesa com exercícios estruturados.

## Níveis

| Nível | Conteúdo |
|-------|---------|
| N5 | Partículas básicas (は, が, を, に, で, と, や, から, まで), ます形, ません形, て形, たる形, なかった形 |
| N4 | たい, ほしい, そうだ, ようだ,  passive, causative, condições (ば, たら, なら, と) |
| N3 | はず, べき, う/よう, 都度, ながら, のに, ために, に基づいて |

## Estrutura dos Exercícios

### 1. Explicação
Breve explicação do ponto gramatical com exemplos.

### 2. Prática
Exercício de completar ou transformar.

### 3. Resposta
Resposta correta com explicação.

### 4. Feedback
Indicar acerto (✓) ou erro com correção.

## Tópicos por Nível

### N5

- Partículas (は, が, を, に, で, と, や, から, まで)
- Verbos no ます形
- Verbos no ない形
- Verbos no て形
- Adjetivos (い/な)
- Demonstativos (これ, それ, あれ, どれ)
- Pronomes pessoais (私, あなた, 彼, 彼女)
- Números
- Tempo (今日, 明日, 昨日, 今)

### N4

- たい / ほしい (querer)
- そうだ (parecer)
- ようだ (parecer)
- Passiva
- Causativa
- Condicionais (ば, たら, なら, と)
- Conjunções (しかし, だから, しかし)
- Comparação (より, ほうが)

### N3

- はず (dever)
- べき (dever)
- う/よう (intenção)
- 都度 (每一次)
- ながら (enquanto)
- のに (apesar de)
- ために (para)
- に基づいて (baseado em)
- によると (segundo)

## Exemplos de Exercícios

### N5: Partícula は vs が

```
Exercício: Complete com は ou が
1) 私___学生です
2) 私___この本を買いません

Respostas:
1) は
2) を

Explicação: は marca o tema, を marca objeto direto
```

### N4: たい vs ほしい

```
Exercício: Escolha entre たい e ほしい
1) 私は映画を___ (assistir)
2) 给我___水を (querer água)

Respostas:
1) みたい ou みたい (quero assistir)
2) ほしい (quero receber)

Explicação: たい = quero fazer (eu como sujeito), ほしい = quero receber
```
