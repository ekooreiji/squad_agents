# Template de Perguntas

Template de perguntas para discovery calls.

---

##Frameworks de Perguntas

### Framework 5W1H

| Pergunta | Uso |
|----------|-----|
| What (O quê) | Entender o pedido |
| Who (Quem) | Identificar stakeholders |
| When (Quando) | Definir timeline |
| Where (Onde) | Contextualizar |
| Why (Por quê) | Entender motivação |
| How (Como) | Definir execução |
