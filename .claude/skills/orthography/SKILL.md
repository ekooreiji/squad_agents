# Orthography

Skill para praticar ortografia japonesa: hiragana e katakana.

## Conteúdo

### Hiragana (ひらがな)
- 46 characteres básicos
- Combinações (ん)
- Duplas (っ)

### Katakana (カタカナ)
- 46 characteres básicos
- Combinações
- Duplas (ッ)

## Modo de Prática

### 1. Leitura
Apresentar carácter e usuário lê.

### 2. Escrita
Ditar leitura e usuário escreve em japanese.

### 3. Ditado
O skill dit日本o e usuário escreve.

## Estrutura dos Exercícios

### Leitura
```
Skill: あめ
Você: ame (chuva)

✓ Correto!
```

### Escrita
```
Skill: "chuva" em hiragana
Você: あめ

✓ Correto!
```

### Ditado
```
Skill: ou
Você: う

✓ Correto!
```

## Progressão

| Fase | Dificuldade |
|------|-------------|
| 1 | hiragana básico |
| 2 | hiragana complexo |
| 3 | katakana básico |
| 4 | katakana complexo |
| 5 | Misturado |

## Mapeamento Hiragana-Katakana

| Hiragana | Katakana | Leitura |
|---------|---------|---------|
| あ | ア | a |
| い | イ | i |
| う | ウ | u |
| え | エ | e |
| お | オ | o |
| か | カ | ka |
| き | キ | ki |
| く | ク | ku |
| け | ケ | ke |
| こ | コ | ko |
| ... | ... | ... |

## Como Usar

1. Escolher modo (leitura/escrita/ditado)
2. Escolher nível (hiragana/katakana/misturado)
3. Praticar com exercícios
4. Receber feedback imediato

## Dicas

- Praticar diariamente
- Escrever à mão
- Criar frases usando os caracteres
