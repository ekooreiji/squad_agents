# Exemplos de Timeline

Exemplos de timelines para diferentes tipos de projeto.

---

## Exemplo 1: Website Simples (4 semanas)

### Timeline

```
Semana 1         Semana 2         Semana 3         Semana 4
|----|----|----|----|----|----|----|----|
[==DISCOVERY==][====DESIGN====][===DEV=====][===LAUNCH==]
Kickoff  Design  Mockups  Dev     Test    Deploy
         Wireframe         QA       Handoff
```

### Detalhamento

| Semana | Atividades | Entregável |
|--------|-----------|------------|
| 1 | Kickoff, pesquisa | Briefing, estratégia |
| 2 | Wireframes, mockups | Home, internas |
| 3 | HTML, CSS, JS | Site funcionando |
| 4 | Testes, ajustes | Go live |

## Exemplo 2: Redesign (8 semanas)

### Timeline

```
Semana 1-2       Semana 3-4       Semana 5-6       Semana 7-8
|----|----|----|----|----|----|----|----|----|----|----|----|----|
[==DISCOVERY===][====DESIGN====][==DEVELOP====][===LAUNCH==]
Audit    Wireframe  Components  Frontend    QA       Deploy
Research Mockups  Design Sys  Backend    UAT
```

### Detalhamento

| Fase | Semanas | Atividades |
|------|---------|-----------|
| Discovery | 1-2 | Auditoria, pesquisa, estratégia |
| Design | 3-4 | Wireframes, mockups, design system |
| Development | 5-6 | Frontend, backend, CMS |
| Launch | 7-8 | QA, UAT, deploy, training |

## Exemplo 4: E-commerce (16 semanas)

### Timeline

```
Semana 1-2     Semana 3-4     Semana 5-8     Semana 9-12    Semana 13-14   Semana 15-16
|----|----|----|----|----|----|----|----|----|----|----|----|----|----|----|----|----|
[==DISCOVERY===][===DESIGN====][===DEV=====][====TEST====][==UAT==][==LAUNCH=]
Research    UX/UI       Core         Testes       Valid       Go live
Strategy    flows       Payments     QA           Training    Marketing
            flows       Shipping     Security
```

### Detalhamento

| Fase | Semanas | Focus |
|------|---------|-------|
| Discovery | 1-2 | Estratégia, arquitetura |
| Design | 3-4 | UX, UI, flows |
| Development | 5-8 | Core platform |
| Test | 9-12 | Testes, QA |
| UAT | 13-14 | Validação, training |
| Launch | 15-16 | Deploy, marketing |
