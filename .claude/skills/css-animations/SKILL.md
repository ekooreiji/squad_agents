# CSS Animations Skill

Skill especializada em animações e transições CSS nativas.

---

## Visão Geral

Esta skill ensina a:

- Criar transições CSS
- Usar keyframes
- Aplicar transforms
- Controlar timing
- Animações responsivas
- Boas práticas de performance
