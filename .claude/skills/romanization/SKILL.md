# Romanization

Skill para praticar romanização (romaji) - conversão entre escrita japonesa e alfabeto romano.

## O que é Romanização

Romanização é a representação da escrita japonesa usando o alfabeto latino (romaji).

## Sistemas

| Sistema | Uso |
|---------|-----|
| Hepburn | Padrão mais usado (tanaka → tanaka) |
| Kunrei | Sistema oficial (tanaka → tanaka) |
| Nihon-shiki | Tradicional |

## Modos de Prática

### 1. Hiragana → Romaji
```
Input: たべる
Output: taberu

✓ ou ✗
```

### 2. Romaji → Hiragana
```
Input: taberu
Output: たべる

✓ ou ✗
```

### 3. Katakana → Romaji
```
Input: ラーメン
Output: ramen

✓ ou ✗
```

### 4. Palavras Completas
```
Input: ありがとうございます
Output: arigatou gozaimasu

✓ ou ✗
```

## Casos Especiais

| Palavra | Romaji | Nota |
|---------|-------|------|
| こんにちは | konnichiwa | o = w |
| おはよう | ohayou | u = w |
| ありがとうございます | arigatou gozaimasu | Compounds |
| チョコレート | chokorīto | Long vowel |
| 游泳池 | yuūchiki | Long vowel |

## Como Usar

1. Escolher modo (hiragana→romaji, romaji→hiragana, katakana, etc)
2. Praticar conversões
3. Receber feedback imediato

## Dicas

- Praticar palavras longas
- Cuidar com w (を, お)
- Longas vogais em katakana
