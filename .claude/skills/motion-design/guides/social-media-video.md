Guia para criar vídeos curtos para redes sociais.

## 2. Estrutura de Vídeo

### 2.1 Abertura (Hook)

```
0-3 segundos
- capture atenção immediately
- usar movimento, cor, som
```

### 2.2 Conteúdo Principal

```
3-30 segundos
- apresentar valor
- manter engajamento
- evitar filler
```

### 2.3 Encerramento (CTA)

```
Últimos 2-3 segundos
- call to action
- encourage interaction
```

---

## 3. Dicas de Produção

### 3.1 Captura de Atenção

| Técnica | Exemplo |
|---------|--------|
| Transições rápido | Cuts every 1-2s |
| Texto grande | Minimum 30px readable |
| Emoji/animation | Adicione movimento |
| Som/hook | Sound grabbing |

### 3.2 Trends

- Use trending sounds (quando aplicável)
- Participate em challenges
- Replicate popular formats

### 3.3 Legibilidade

- Texto always large
- Font simples, bold
- High contrast
- Consider sound OFF viewing
