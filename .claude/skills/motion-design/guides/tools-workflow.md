## 2. Fluxo de Trabalho

### 2.1 Processo

```
1. Brief
   - Definir objetivos
   - Entender público
   │
2. Research
   - Referências visuais
   - Estudo de motion trends
   │
3. Storyboard
   - Desenhar frames key
   - Definir timing
   │
4. Asset Creation
   - Criar elementos graphics
   │
5. Animação
   - Aplicar motion
   │
6. Revisão
   - Testar timing
   - Ajustar easing
   │
7. Export
   - Escolher formato
   - Otimizar
```

### 2.2 Timeline Structure

| Fase | Tempo | Descrição |
|------|-------|-----------|
| Planning | 10% | Brief, research |
| Storyboard | 15% | Frames key |
| Execution | 60% | Animação |
| Refinement | 10% | Ajustes |
| Export | 5% | Formatos |

## 3. Software Basics

### 3.1 After Effects Basics

```
1. New Project
2. Create Composition (preset: 1920x1080, 30fps)
3. Import Assets
4. Create Timeline
5. Add Keyframes
6. Apply Easing
7. Export (Media Encoder)
```

### 3.2 Key Shortcuts (AE)

| Atalho | Função |
|--------|--------|
| Space | Play/Pause |
| J/K/L | Rewind/Forward/Pause |
| N | End of work area |
| I | Start of work area |
| M | Markers de propriedades |
| G | Track matte |
