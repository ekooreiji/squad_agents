# 12 Animation Principles

Guia dos 12 princípios de animação da Disney adaptados para motion design.

---

## Os 12 Princípios

### 1. Squash and Stretch

```
════════╱═══════
```
- Adiciona flexibilidade e peso
- Use em: buttons, loading states

### 2. Anticipation

```
← ───━╱
```
- Prepara para ação
- Use em: button clicks, page transitions

### 3. Staging

```
    ═══════━
    ↑
    ════════
```
- Apresenta ideia principal
- Use em: hero animations, content reveals

### 4. Straight Ahead / Pose to Pose

| Abordagem | Uso |
|----------|-----|
| Straight | Fluid, unpredictable |
| Pose to Pose | Cinematic, controlled |

### 5. Follow Through / Overlapping Action

```
movement ═══════→   overshoot ╱↘
```
- Partes se movem em momentos diferentes
- Use em: elementos que se arrastam

### 6. Slow In / Slow Out

```
    ╱═══╲
```
- Acelera e desacelera
- Use em: movements naturais

### 7. Arc

```
 ╱
╱
```
- Movimentos curvados
- Evite linhas retas artificiais

### 8. Secondary Action

```
Main ══════════→
              ↘
Secondary ──────────────→
```
- Suporta ação principal
- Use em: elementos de suporte

### 9. Timing

| Sensação | Frames |
|----------|--------|
| Pesado | 24 fps |
| Leve | 12 fps |

### 10. Exaggeration

```
Normal    ════
Exaggerated ═══════════
```

### 11. Solid Drawing

- Avoid "floaty" animations
- Consider weight and physics

### 12. Appeal

- Carismático e atraente
- Design amigável

---

## Adaptation para UI

| Princípio | Aplicação UI |
|----------|---------------|
| Squash/Stretch | Button feedback |
| Anticipation | Pré-visualização de transição |
| Staging | Hero animations |
| Timing | Velocidade de transição |
| Ease-in-out | Curvas de easing |
