# Introdução ao Spectre.css

Framework CSS moderno e leve.

---

## O que é Spectre.css

Framework CSS moderno:
- ~10KB total
- Flexbox e Grid
- v0.5.3

---

## Instalação

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/spectre.css@0.5.3/dist/spectre.min.css">
```

---

## Diferenças

| Aspecto | Spectre | Bootstrap |
|--------|--------|----------|
| Tamanho | ~10KB | ~300KB |
| Base | Moderno | Tradicional |
| CSS Variables | Sim | Parcial |

---

## Quick Start

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Spectre.css</title>
    <link rel="stylesheet" href="spectre.min.css">
</head>
<body>
    <div class="container">
        <h1>Olá Spectre</h1>
    </div>
</body>
</html>
```
