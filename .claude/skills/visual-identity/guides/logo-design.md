# Logo Design

Guia completo para criação de logotipos e símbolos de marca.

---

## 1. Tipos de Logotipos

### 1.1 Por Estrutura

| Tipo | Exemplo | Uso |
|------|---------|-----|
| Wordmark | Google | Nomes curtos |
| Lettermark | IBM, NASA | Siglas |
| Pictorial | Apple | Ícones |
| Abstract | Nike | Símbolos abstratos |
| Mascot | Michelin | Personagens |
| Combination | Adidas | Ícone + word |

### 1.2 Por Estilo

| Estilo | Características | Exemplo |
|--------|-----------------|----------|
| Minimal | Simples, limpo | Stripe |
| Line | Contornos | Starbucks |
| Flat | Sólido, sem gradiente | Spotify |
| 3D | Profundidade | Discord |
| Hand-drawn | Orgânico | Cafepress |

### 2.2 Memorabilidade

- Formas únicas

- Funciona em diferentes tamanhos
- Funciona em diferentes cores
- Funciona em diferentes fundos

## 3. Processo de Criação

### 3.1 Fases

```
FASE 1: Brief
        - Entender marca
        - Definir público
        - Competidores
        
FASE 2: Pesquisa
        - Moodboard
        - Referências
        - Tendências
        
FASE 3: Concepção
        - Sketchs iniciais
        - Explorações
        - Iterações
        
FASE 4: Refinamento
        - Selección
        - Polimento
        - Variações
        
FASE 5: Entrega
        - Versões finais
        - Arquivos
        - Manual
```

### 3.2 Ferramentas

| Fase | Ferramenta |
|------|-----------|
| Sketch | Pencil, Procreate |
| Vetor | Illustrator, Inkscape |
| Export | Figma, Sketch |

## 4. Versões do Logo

### 4.1 Por Tamanho

| Versão | Uso |
|--------|-----|
| Full | Header principal |
| Small | Favicon,社交媒体头像 |
| Simplified | Impressos pequenos |

### 4.2 Por Cor

| Versão | Uso |
|--------|-----|
| Full color | Original |
| Monochrome | Umacolor |
| Black | Impressão |
| White | Fundos escuros |

### 4.3 Por Contexto

```
┌─────────────────┐
│   Horizontal    │  ← Uso principal
├─────────────────┤
│                 │
│      Ícone      │  ← Redes sociais
│                 │
├─────────────────┤
│     Stacked     │  ← Espaçosquadrados
└─────────────────┘
```

### 5.1 Formatos

| Formato | Extensão | Uso |

| SVG | .svg | Web |
| PNG | .png | Web transparency |
| PDF | .pdf | Documentos |

### 5.2 Cores

- CMYK para impressão
- RGB para digital
- Pantone para referência
