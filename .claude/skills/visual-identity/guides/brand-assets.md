# Brand Assets

Guia para gestão e organização de assets de marca.

---

## 1. O que são Assets

Assets são todos os arquivos visuais da marca: logos, ícones, fontes, imagens, templates.

---

## 2. Tipos de Assets

### 2.1 Categorias

| Categoria | Exemplos |
|-----------|---------|
| **Logos** | .ai, .eps, .svg, .png |
| **Fontes** | .ttf, .otf, .woff |
| **Cores** | Palettes, swatches |
| **Ícones** | Icon set |
| **Templates** | Apresentações, documentos |
| **Imagens** | Fotos, ilustrações |

---

---

## 4. Nomenclatura

### 4.1 Padrão

```
[asset]-[versão]-[cor]-[tamanho].[ext]

Exemplos:
logo-full-color.svg
logo-black-500px.png
icon-add-24px.svg
```

### 4.2 Versionamento

```
v1.0 - Lançamento
v1.1 - Correções
v2.0 - Rebranding
```

## 5. Distribuição

### 5.1 Locais

| Local | Acesso |
|-------|--------|
| Brand folder compartilhado | Team |
| Brand portal | Company |
| Downloads page | Público |

### 5.2 Permissões

| Asset | Interno | Externo |
|--------|---------|---------|
| Logos | ✓ | ✓ (limitado) |
| Cores | ✓ | ✓ |
| Tipografia | ✓ | ✗ (licença) |
