# Foundation CSS Skill

Skill especializada em Foundation CSS para criar interfaces modernas e responsivas.

---

## Visão Geral

Esta skill ensina a:

- Configurar e usar Foundation
- Implementar XY Grid
- Criar componentes semânticos
- Desenvolver formulários com Abide
- Aplicar tipografia
- Customizar com Sass
- Criar designs responsivos

## Diferenças Foundation vs Bootstrap

| Aspecto | Foundation | Bootstrap |
|--------|------------|-----------|
| Grid | XY Grid flex | 12-col fixed |
| Semantic | Sim | Parcial |
| Menu off-canvas | Sim | Não |
| Validação forms | Abide | HTML5 native |
