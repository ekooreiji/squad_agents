# Template de Script de Teste de Usabilidade

Template completo para conduzir testes de usabilidade.

---

## Estrutura do Script

```markdown
# Script: Teste de Usabilidade - [Nome do Produto]

## Informações Gerais

| Campo | Valor |
|-------|-------|
| Data | [Data] |
| Local | [Local/Remoto] |
| Moderador | [Nome] |
| Observador | [Nome] |
| Duração estimada | [X] min |

## Participante

| Campo | Valor |
|-------|-------|
| ID | P[X] |
| Perfil | [Perfil] |
| Experiência | [Nível] |
| Dispositivo | [Tipo] |

## Tarefas

### Tarefa 1: [Nome]

| Campo | Descrição |
|-------|-----------|
| Objetivo | [O que completar] |
| Cenário | [Contexto] |
| Pré-condição | [O que precisa antes] |

**Instrução para participante:**
> "[Ler para o participante]"

**Sucesso se:** [Condição]

---

## Roteiro Completo

### Abertura (5 min)

```
[APRESENTAÇÃO]
Olá, obrigado por participar. Vou pedir para você realizar algumas tarefas e pensar 
alto enquanto faz. Não tem resposta certa ou errada - queremos observar como você 
usa o produto.

[CONSENTIMENTO]
Posso gravar a tela e áudio? Você pode parar a qualquer momento.

[PERGUNTAS AQUECIMENTO]
- Quanto você usa [produto]?
- Como você usually faz [tarefa relacionada]?
```

### Tarefas (30-45 min)

```
[TAREFA 1]
[Instrução]

[Pausa para participant]

[Se precisar de ajuda:]
"Pode pedir ajuda se tiver dificuldades, mas preferimos que tente sozinho(a)."

[Perguntas pós-tarefa:]
- Como foi?
- Teve alguma dúvida?
- O que esperava que acontecesse?
```

### Questionário (5 min)

```
[PERGUNTAS DE SATISFAÇÃO]
- Em uma escala de 1 a 5, quão fácil foi completar a tarefa?
- O que foi mais difícil?
- O que poderia melhorar?

[System Usability Scale - SUS]
[Avalie de 1 (discordo totalmente) a 5 (concordo totalmente)]
1. Eu acho que gostaria de usar este sistema frequentemente.
2. Eu achei o sistema desnecessariamente complexo.
...
```

### Encerramento (5 min)

```
[AGRADECIMENTO]
Obrigado pela participação! Seu feedback é muito valioso.

[PERGUNTAS FINAIS]
- Algo mais que gostaria de adicionar?
- Vocêindicaria este produto para alguém?
```
