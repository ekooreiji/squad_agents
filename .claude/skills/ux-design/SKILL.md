# UX Design Skill

Skill especializada em experiência do usuário, englobando pesquisa, criação de personas, fluxos, testes de usabilidade e arquitetura da informação.

---

## Visão Geral

Esta skill oferece guias, templates e ferramentas para criar experiências de usuário centrado no usuário (User-Centered Design).

---

## Guias por Categoria

### Pesquisa e Descoberta
- `guides/user-research.md` - Métodos de pesquisa com usuários
- `guides/personas.md` - Criação e validação de personas

### Design de Interação
- `guides/user-flows.md` - Mapeamento de fluxos de usuário
- `guides/interaction-design.md` - Princípios de design de interação
- `guides/information-architecture.md` - Arquitetura da informação

### Validação
- `guides/usability-testing.md` - Testes de usabilidade

## O que esta skill pode fazer

| # | Ação | Descrição |
|---|------|------------|
| 1 | Conduzir pesquisa com usuários | Entrevistas, surveys, observação |
| 2 | Criar personas | Perfis de usuários alvo |
| 3 | Mapear user flows | Fluxos de interação |
| 4 | Executar testes de usabilidade | Validação com usuários reais |
| 5 | Definir arquitetura da informação | Organização de conteúdo |
| 6 | Realizar auditoria UX | Avaliação heurística |

## Boas Práticas

- Sempre validar com usuários reais
- Iterar baseado em feedback
- Documentar descobertas
- Manter foco no usuário, não no produto
- Usar dados analytics para complementar pesquisa
