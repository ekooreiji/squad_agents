# Profiling

## O que é Profiling

Profiling é a análise de performance para identificar gargalos.

## Tipos de Profiling

| Tipo | Descrição |
|------|-----------|
| **CPU** | Tempo de CPU por função |
| **Memory** | Uso de memória |
| **I/O** | Operações de disco/rede |
| **Line-by-line** | Tempo por linha |

## Por que fazer

- Identificar gargalos reais
- Otimizar onde importa
- Evitar otimização prematura
