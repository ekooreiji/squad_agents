# Wireframe Template

## Visão Geral

Template para criação de wireframes de baixa fidelidade.

---

## Estrutura do Wireframe

```
┌─────────────────────────────────────────────┐
│                    HEADER                  │
│  [Logo]    [Nav]    [Search]    [User]   │
├─────────────────────────────────────────────┤
│                                             │
│                  HERO SECTION               │
│         [Título] + [Descrição]            │
│              [CTA Button]                 │
│                                             │
├─────────────────────────────────────────────┤
│                                             │
│              CONTENT GRID                   │
│  ┌─────┐  ┌─────┐  ┌─────┐              │
│  │Card │  │Card │  │Card │              │
│  └─────┘  └─────┘  └─────┘              │
│                                             │
├─────────────────────────────────────────────┤
│                    FOOTER                   │
│   [Links]  [Social]  [Copyright]         │
└─────────────────────────────────────────────┘
```

---

## Elementos do Wireframe

### Header
| Elemento | Descrição |
|----------|-----------|
| Logo | Identidade visual |
| Navigation | Links principais |
| Search | Barra de busca |
| User | Menu do usuário |

### Hero Section
| Elemento | Descrição |
|----------|-----------|
| Title | Título principal |
| Subtitle | Descrição breve |
| CTA | Botão de ação |

### Content Grid
| Elemento | Descrição |
|----------|-----------|
| Cards | Conteúdo em grid |
| Images | Representação visual |
| Text | Lipsum简短 |

---

## Anotações

### Legenda de Elementos
| Símbolo | Significado |
|---------|-------------|
| [X] | Botão clicável |
| ≡ | Menu |
| → | Link/Navegação |
| () | Imagem |
| T | Texto |

---

## Boas Práticas

- Usar formas simples
- Manter espaçamento consistente
- Não usar cores detalhadas
- Focar em estrutura
- Anotar interações
