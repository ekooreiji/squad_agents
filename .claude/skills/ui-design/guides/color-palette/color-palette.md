## Contraste WCAG

### Requisitos Mínimos
| Nível | Texto Normal | Texto Grande |
|-------|--------------|---------------|
| AA | 4.5:1 | 3:1 |
| AAA | 7:1 | 4.5:1 |

### Verificação de Contraste
- Use ferramentas como WebAIM Contrast Checker
- Teste todas as combinações de cor de fundo e texto

## Boas Práticas

- Limite a 3-5 cores principais
- Use cores neutras para equilibrar
- Garanta contraste mínimo WCAG AA
- Considere daltonismo
- Teste em diferentes dispositivos
