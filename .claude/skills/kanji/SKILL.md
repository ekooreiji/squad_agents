# Kanji

Skill para estudo de kanji japanese conforme o nível Jōyō.

## Níveis Jōyō

| Nível | Kanji | Descrição |
|-------|------|-----------|
| N5 | 80 | Kanji básicos para leitura básica |
| N4 | 160 | Kanji para vida diária |
| N3 | 213 | Kanji intermediários |
| N2 | 1000 | Kanji avançados |
| N1 | 2000+ | Kanji de leitura fluente |

## Estrutura do Kanji

```
[Kanji]
Leitura: [on'yomi] / [kun'yomi]
Significado: [significado]
Exemplo: [palavra usando o kanji]
```

## Exemplo de Kanji

### 火 (か / ひ)
- Leitura: カ (on), ひ (kun)
- Significado: Fogo
- Exemplo: 火曜日 (かようび) = terça-feira

### 水 (すい / みず)
- Leitura: スイ (on), みず (kun)
- Significado: Água
- Exemplo: 水曜日 (すいようび) = quarta-feira

### 人 (じん / にん / ひと)
- Leitura: ジン, ニン (on), ひと (kun)
- Significado: Pessoa, gente
- Exemplo: 日本人 (にほんじん) = japonês (pessoa)

## Modos de Prática

### 1. Flashcard
Apresentar kanji → usuário lembra leitura e significado.

### 2. Leitura
Apresentar palavra → usuário lê em hiragana.

### 3. Escrita
Definir palavra → usuário escreve o kanji.

### 4. Significado
Apresentar kanji → usuário indica o significado.
