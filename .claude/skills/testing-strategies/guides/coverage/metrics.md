# Coverage Metrics

## Tipos de Coverage

| Tipo | Descrição |
|------|-----------|
| **Line** | % de linhas executadas |
| **Branch** | % de branches executadas |
| **Function** | % de funções chamadas |
| **Statement** | % de statements |

## Targets

| Nível | Target |
|-------|--------|
| **Unit** | 80-90% |
| **Integration** | 70-80% |
| **E2E** | 50-60% |

## Coverage por Tipo de teste

```
Total: 70%
├── Unit: 80%
├── Integration: 60%
└── E2E: 40%
```

## Interpretação

| Coverage | Significado |
|---------|-----------|
| < 50% | Pouco coberto |
| 50-70% | Razoável |
| 70-80% | Bom |
| 80-90% | Muito bom |
| 90%+ | Excelente |
